## Purpose
Criar roteiro completo para vídeo curto de afiliado (15-90 segundos) seguindo a estrutura Hook → Problem → Agitate → Solution → CTA, com notas visuais para cada segmento e múltiplas opções de execução.

## Prerequisites
- Hook definido ou set de hooks criado (`create-hook-set.md` executado)
- Produto afiliado ou nicho definido
- Plataforma de publicação definida (TikTok, Reels, Shorts, YouTube)
- Duração alvo definida (15s / 30s / 60s / 90s)
- Tom definido (educativo, entretenimento, motivacional, drama, humor)

## Steps

1. **Definir a duração e distribuição de tempo** — Com base na duração alvo, distribuir os segundos entre cada seção do roteiro (Hook, Problem, Agitate, Solution, CTA).

2. **Selecionar o hook** — Escolher o hook do set criado anteriormente, ou criar um novo alinhado com o conteúdo do vídeo.

3. **Escrever o segmento Problem (3-8s)** — Nomear o problema específico que o vídeo resolve. Deve ser identificável pelo avatar em menos de 3 segundos.

4. **Escrever o segmento Agitate (8-15s)** — Amplificar a dor do problema. Tornar o problema "real" na mente do espectador. Usar consequências específicas, não genéricas.

5. **Escrever o segmento Solution (15-40s)** — Apresentar a solução (método, produto, framework). Mostrar o resultado — não apenas descrever o processo. Ser específico e acionável.

6. **Escrever o CTA (40-60s)** — Uma ação clara e simples. Para afiliados: "Link na bio", "Me segue para mais", "Comentar 'X' para receber" ou "Acessa o link para...".

7. **Adicionar notas visuais** — Para cada segmento, descrever o que aparece na câmera: posicionamento, texto on-screen, B-roll, expressão/energia.

8. **Criar versão alternativa (A/B)** — Escrever uma segunda versão com o mesmo conteúdo mas hook diferente ou ordem de revelação diferente, para teste.

9. **Documentar o roteiro** — Salvar no arquivo de output com todas as versões e notas de produção.

### Regras de CTA para Afiliados

```
O QUE FAZER:
✓ "Link na bio" para produtos externos (afiliado não pode linkar diretamente no TikTok)
✓ "Comenta 'X' para receber" → gera engajamento + DM automation
✓ "Me segue para mais sobre [nicho]" → construção de audiência
✓ Mostrar visualmente onde clicar (apontar, seta, highlight)

O QUE EVITAR:
✗ Múltiplos CTAs no mesmo vídeo (confunde e dilui ação)
✗ CTA genérico ("clique aqui") sem contexto de benefício
✗ Pedir clique E follow E comentário no mesmo vídeo
✗ Affiliate disclosure ausente se vídeo promove produto diretamente
```

## Veto Conditions
- Vídeo que começa com "Olá, meu nome é..." → BLOQUEAR (introduções matam o watch time — começar direto no hook)
- Solution antes do Problem/Agitate → ALERTAR (solução sem contexto de dor não ressoa — seguir a estrutura)
- CTA com mais de uma ação → ALERTAR (1 vídeo = 1 CTA = 1 ação)
- Duração acima de 90s sem justificativa de formato longo → ALERTAR (vídeos curtos têm maior completion rate e distribuição algorítmica)
- Afirmação de resultado específico sem evidência visual → ALERTAR (claims sem prova são desconfiáveis e podem violar políticas da plataforma)
- Script sem notas visuais → ALERTAR (executar sem visual definido resulta em produção aleatória)
