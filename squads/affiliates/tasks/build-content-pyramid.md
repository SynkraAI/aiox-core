## Purpose
Construir a Content Pyramid da operação de afiliado seguindo o modelo GaryVee: a partir de uma peça de conteúdo longo (pillar), extrair 10+ micro-conteúdos nativos para cada plataforma, maximizando o alcance com o mínimo de esforço de criação.

## Prerequisites
- Conteúdo longo (pillar) gravado ou publicado (YouTube, podcast, live, webinar, ou artigo 2.000+ palavras)
- Plataformas ativas mapeadas
- Calendário editorial ativo (`design-social-calendar.md` executado)
- Ferramenta de edição disponível (CapCut, DaVinci Resolve, Canva, ou editor contratado)
- Produto(s) afiliado(s) para integrar naturalmente no conteúdo

## Steps

1. **Definir ou selecionar o pillar content** — Escolher a peça de conteúdo longo que será a base da pirâmide. Deve ser: rico em informação, com pelo menos 5 ângulos extraíveis, e conectado ao nicho do produto afiliado.

2. **Mapear os 10+ micro-ângulos** — Assistir/ler o pillar completo e anotar cada insight, momento memorável, citação, dado ou ponto que pode virar uma peça independente.

3. **Planejar a distribuição por plataforma** — Alocar cada micro-ângulo à plataforma mais adequada para aquele tipo de conteúdo, respeitando os formatos nativos.

4. **Criar os micro-conteúdos** — Produzir cada peça de conteúdo com a especificação de plataforma correta. Para vídeo: cortes do pillar ou gravações novas baseadas nos ângulos. Para texto: threads, carrosséis, posts.

5. **Integrar o produto afiliado naturalmente** — Em 1 de cada 4 micro-conteúdos, incluir menção ou CTA para o produto afiliado de forma orgânica (não interruptiva).

6. **Planejar a publicação ao longo de 2-4 semanas** — Distribuir os 10+ micro-conteúdos pelo calendário editorial para criar consistência de presença sem precisar criar novo pillar toda semana.

7. **Documentar a pirâmide** — Registrar a estrutura completa no arquivo de output com a lista de todos os micro-conteúdos planejados.

## Framework

### A Content Pyramid — Estrutura GaryVee

```
                    ┌─────────────────────────────────┐
                    │      PILLAR CONTENT (TOPO)      │
                    │  YouTube Long-form / Podcast /  │
                    │  Live / Webinar / Artigo 2k+   │
                    │  1x por semana (ou 1-2x/mês)   │
                    └──────────────┬──────────────────┘
                                   │
                    ┌──────────────┴──────────────────┐
                    │     DERIVATIVE CONTENT          │
                    │   Extraído diretamente do Pillar│
                    └──────────────┬──────────────────┘
                                   │
         ┌─────────────────────────┼─────────────────────────┐
         │                         │                         │
    ┌────┴─────┐             ┌─────┴────┐             ┌──────┴────┐
    │  VIDEO   │             │   TEXT   │             │  VISUAL   │
    │  CLIPS   │             │ CONTENT  │             │ CONTENT   │
    │ 3-5 cuts │             │ 3-5 posts│             │ 2-3 posts │
    └──────────┘             └──────────┘             └───────────┘
         │                         │                         │
    TikTok/Reels             Twitter Thread            Carrossel
    YouTube Shorts           LinkedIn Post             Infográfico
    Stories (vídeo)          Newsletter                Quote card
```

### Mapa de Extração de 10 Micro-Conteúdos

```
A partir de 1 pillar (ex: "Como Afiliados de Fitness Faturam R$5k/mês"):

VÍDEOS (4-5 cortes):
  V1: Corte do melhor momento (citação memorável) → TikTok/Reels/Shorts
  V2: Corte da maior revelação (epiphany moment) → TikTok/Reels
  V3: Corte do case study específico → YouTube Shorts
  V4: Corte da sessão de Q&A (se live) → Stories
  V5: Trailer/teaser do pillar completo → Todos

TEXTO (3-4 posts):
  T1: Thread "10 insights da [live/video/podcast]" → Twitter/X / LinkedIn
  T2: Newsletter com resumo + link para pillar → Email list
  T3: Post LinkedIn sobre a lição principal → LinkedIn
  T4: Caption longa no Instagram com insight + carrossel → Feed

VISUAL (2-3 posts):
  I1: Quote card do melhor momento → Stories / Feed
  I2: Carrossel "5 pontos que aprendi" → Feed / LinkedIn
  I3: Infográfico do processo/framework ensinado → Feed / Pinterest

BONUS:
  B1: Poll/enquete baseada em pergunta levantada no pillar → Stories
  B2: "Parte 2" respondendo comentários do pillar → TikTok/Reels
```

### Integração do Produto Afiliado na Pirâmide

```
Regra: 1 de cada 4 micro-conteúdos menciona o produto (ratio 3:1)

FORMAS NATURAIS DE INTEGRAR:
  ✓ "Para implementar o que ensino nesse video, o produto X foi essencial"
  ✓ "Case study: como [nome] usou [produto] para alcançar [resultado do pillar]"
  ✓ "A ferramenta que mostro no pillar: link na bio"
  ✓ Carrossel com o passo a passo + último slide com produto como "próximo passo"

FORMAS FORÇADAS (evitar):
  ✗ Inserir promoção no meio de conteúdo educativo sem conexão lógica
  ✗ "Ah, e por falar nisso, precisa comprar [produto]..."
  ✗ Dedicar posts de valor a falar de features do produto em vez de benefícios
```

### Tempo de Vida de Cada Tipo de Conteúdo

| Plataforma | Vida Útil do Post | Implicação |
|-----------|------------------|-----------|
| TikTok | 3-14 dias (viral) / 0-2 dias (normal) | Publicar em janelas de pico |
| Instagram Feed | 24-48h (algoritmo) | Primeiras 2h são críticas |
| YouTube Shorts | Semanas a meses (descoberta) | Otimizar título e thumbnail |
| Twitter/X | 15-60 minutos | Publicar em pico de audiência |
| YouTube Long-form | Meses a anos (SEO) | Investir em SEO do vídeo |

### Workflow de Produção da Pirâmide

```
DIA 1 (Produção do Pillar):
  - Gravar/publicar o pillar completo
  - Listar os momentos-chave durante a gravação/edição

DIA 2 (Extração de clips):
  - Identificar 5 cortes no pillar
  - Editar: adicionar legendas, texto on-screen, intro/outro removidos

DIA 3 (Criação de texto e visual):
  - Escrever thread / post LinkedIn
  - Criar carrossel no Canva
  - Criar quote cards

DIAS 4-21 (Publicação distribuída):
  - Publicar 1-2 peças por dia
  - Monitorar métricas e ajustar sequência
  - Responder comentários (aumenta distribuição)
```

## Veto Conditions
- Criar micro-conteúdo sem pillar definido (conteúdo solto sem âncora) → ALERTAR (sem pillar, a pirâmide não tem base e o conteúdo perde coerência temática)
- Republicar cortes do pillar sem adaptação para a plataforma → ALERTAR (corte bruto de YouTube como TikTok sem edição nativa tem performance 60-80% menor)
- Mais de 30% dos micro-conteúdos sendo pitch de produto → ALERTAR (pirâmide vira campanha de ads — audiência abandona)
- Pillar content com menos de 15 minutos → ALERTAR (pouco material para extrair 10+ micro-conteúdos com qualidade — considerar juntar 2 pillars ou aumentar duração)
- Micro-conteúdos todos publicados no mesmo dia → ALERTAR (saturação e sem consistência semanal — distribuir ao longo de 2-3 semanas)
