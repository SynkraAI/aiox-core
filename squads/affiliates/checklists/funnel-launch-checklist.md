# Funnel Launch Checklist

> Verify every component of the affiliate funnel is functional, compliant, and optimized before driving traffic.

**Fase:** Funnels
**Usado por:** `workflows/funnel-deploy.md`, `tasks/funnel-qa.md`
**Items:** 9

---

## Checklist

- [ ] **Opt-in page live with headline + benefits** — Opt-in page is publicly accessible; headline clearly states the value proposition; minimum 3 specific benefits are listed; form captures email at minimum
- [ ] **Lead magnet deliverable tested** — Lead magnet (PDF, video, mini-course, etc.) is correctly delivered after opt-in; download link or access email sends within 5 minutes; file opens without errors
- [ ] **Thank you page redirect working** — Post-opt-in redirect lands on correct thank-you page; page loads without errors; confirmation message matches what was promised on opt-in page
- [ ] **Bridge page with story + credibility + CTA** — Bridge page includes personal story or problem framing, credibility element (testimonial, result, or authority signal), and a single clear CTA pointing to affiliate offer
- [ ] **Affiliate disclosure present** — Disclosure statement is visible on all funnel pages that contain or lead to affiliate links; placement is before any monetized content, not buried in footer
- [ ] **Mobile responsive (test on 3 devices)** — Funnel tested on at least 3 device types (e.g., iPhone, Android phone, tablet); all CTAs are tappable, text is readable, no horizontal scroll
- [ ] **Tracking UTM configured** — All funnel entry links include UTM parameters (source, medium, campaign minimum); UTM data flows correctly into analytics platform and can be segmented
- [ ] **Email integration connected** — Opt-in form successfully writes to email service provider list; subscriber receives welcome sequence within configured delay; test subscriber visible in ESP dashboard
- [ ] **Page load < 3 seconds** — All funnel pages load in under 3 seconds on a simulated 4G connection via PageSpeed Insights or GTmetrix; images are compressed and above-the-fold content is prioritized
