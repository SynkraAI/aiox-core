## How to Use This Template

1. Create all 5 hooks for a single video topic
2. Post using Hook 1 first (your best hypothesis)
3. If it underperforms (< 40% 3-sec view rate after 24h), swap to Hook 2
4. Document results below — learn which hook archetypes work for your audience
5. Winning hook types inform future content planning
