# SOP Factory

Elite Standard Operating Procedures squad. Creates, extracts, analyzes, grades and converts SOPs using world-class process documentation methodologies.

## What It Does

Transforms knowledge into SOPs for humans and AI agents, based on Deming, Gawande, Toyota, ISO 9001, FDA/GMP, and Six Sigma.

## Agents

| Agent | Role |
|-------|------|
| `sop-chief` | Squad orchestrator and lifecycle coordinator |
| `sop-analyst` | Analyzes and grades existing SOPs |
| `sop-creator` | Creates human-readable SOPs |
| `sop-ml-architect` | Creates AI/ML-consumable SOPs (YAML/JSON) |
| `sop-extractor` | Extracts SOPs from existing processes |
| `sop-auditor` | Quality gate and compliance validation |

## Workflows

1. **SOP Creation Pipeline** — gather requirements > draft > review > refine > validate > publish
2. **SOP Audit Pipeline** — ingest > analyze > score > benchmark > remediate > certify
3. **SOP Extraction Pipeline** — capture input > identify steps > structure > validate > format > publish
