# Checklist: Synthesis Quality

Quality gate para a fase de síntese (synthesize-mind). Valida que o DNA combinado é coerente e completo.

---

## DNA Mental Coverage (BLOCKING)

- [ ] Layer 1 — Behavioral Patterns (writing_style + tone.by_context)
- [ ] Layer 2 — Communication Style (vocabulary + storytelling + tone)
- [ ] Layer 3 — Routines & Habits (recognition_patterns)
- [ ] Layer 4 — Recognition Patterns (instant_detection)
- [ ] Layer 5 — Mental Models (frameworks + heuristics)
- [ ] Layer 6 — Values Hierarchy (immune_system + fierce_defenses)
- [ ] Layer 7 — Core Obsessions (anti_patterns + attention_triggers)
- [ ] Layer 8 — Productive Paradoxes (contradictions + tensions)

**Minimum:** 6/8 layers covered → PASS
**Below 6/8:** VETO — re-extract missing layers before proceeding

---

## Trindade Sagrada (AN003)

- [ ] **Playbook** presente (how-to instructions, principles)
- [ ] **Framework** presente (IF/THEN heuristics, decision pipeline)
- [ ] **Swipe File** presente (recurring stories, examples, case studies)

**All 3 required.** Missing any → FLAG, identify which extraction phase to re-run.

---

## Stage Architecture (AN004)

- [ ] Stage evaluation performed (consulted clone-stages-guide.md)
- [ ] If has_stages=true: max 5 stages defined
- [ ] Each stage has: trigger, behavior, voice_overrides, transitions
- [ ] Transitions are bidirectional where appropriate

---

## Cross-Reference Integrity (AN007)

- [ ] Voice-Thinking alignments documented (min 3)
- [ ] Voice-Thinking tensions documented (min 1)
- [ ] Reinforcement patterns documented (min 2)
- [ ] Cross-references are bidirectional (Voice cites Thinking AND vice versa)

---

## Coherence Check

- [ ] Overall coherence score assigned (X/10)
- [ ] Identity statement aligns with both Voice and Thinking DNA
- [ ] No contradictions between Voice anti-patterns and Thinking anti-patterns
- [ ] Immune system is consistent across Voice (rejections) and Thinking (vetos)

---

## Output Validation

- [ ] mind_dna_complete.yaml generated using template
- [ ] quality_dashboard.md generated using template
- [ ] Fidelity level calculated and justified
- [ ] All metadata fields populated (expert_name, domain, date, version)

---

**Scoring:** Count checked items / total items.
- >= 90%: PASS
- 75-89%: PASS with notes
- < 75%: VETO — address gaps before proceeding to smoke test
