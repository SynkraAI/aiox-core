## 🎯 Trilhas Disponíveis

### 🏃 Express (8-12h) - Para Eventos Urgentes
```
Módulo 1 (Resumido 2h) → Módulo 2 (Resumido 2h) → Módulo 3 (Resumido 3h) → Prática Intensiva (3-5h)
│                         │                         │                        │
└─ Abertura criativa     └─ Venda resultado       └─ Estrutura 3-5 passos   └─ Palestra 20-30 min
```
**Entrega:** Palestra técnica funcional de 20-30 min (sem história pessoal)

---

### 🚶 Iniciante (15-20h) - Fundamentos
```
Módulo 1 (6-7h) → Módulo 2 (7-9h) → Módulo 3 (9-12h) → Prática (5-7h)
│                  │                 │                   │
└─ Conexão        └─ Promoção      └─ Entrega         └─ Integração
   inicial           conteúdo         estruturada
```
**Entrega:** Palestra técnica de 20 min com abertura, promoção e conteúdo estruturado

---

### 🏃 Intermediário (25-30h) - Storytelling Completo
```
Revisão M1-3 (3-4h) → Módulo 4 (12-15h) → Módulo 5 (7-10h) → Integração (8-10h)
│                      │                   │                   │
└─ Fundamentos        └─ História         └─ Vídeo           └─ Palestra completa
   (se não fez)          autêntica           recapitulativo      40 min + vídeo
```
**Entrega:** Palestra completa de 40 min com todos os 5 passos + vídeo final

---

### 🏆 Master (50-60h) - Domínio Completo + Evento Real
```
Módulos 1-5 (30-35h) → Projeto Final (20-25h)
│                       │
│                       ├─ Palestra 60 min
│                       ├─ Evento real (50+ pessoas)
│                       ├─ Gravação profissional
│                       └─ Feedback da audiência
│
└─ Todos os conceitos dominados
```
**Entrega:** Palestra de 60 min em evento profissional real, gravada, com aplauso de pé

## 🗓️ Cronogramas Sugeridos

### 🏃 Intensivo (2 semanas)
```
Semana 1: Módulos 1-3 (7h/dia)
Semana 2: Módulos 4-5 + Projeto (7h/dia)
Total: 14 dias x 7h = 98h (inclui prática)
```

### 🚶 Moderado (1 mês)
```
Semana 1: Módulos 1-2 (3-4h/dia)
Semana 2: Módulo 3 (3-4h/dia)
Semana 3: Módulo 4 (3-4h/dia)
Semana 4: Módulo 5 + Projeto (3-4h/dia)
Total: 4 semanas x 5 dias x 3.5h = 70h
```

### 🐢 Confortável (3 meses)
```
Mês 1: Módulos 1-2 (2h/dia, 3x/semana)
Mês 2: Módulos 3-4 (2h/dia, 3x/semana)
Mês 3: Módulo 5 + Projeto (2h/dia, 3x/semana)
Total: 12 semanas x 3 dias x 2h = 72h
```

## 🔄 Recomendações por Perfil

### 👤 Nunca Dei Palestra
**Trilha:** Iniciante (15-20h)
**Cronograma:** Confortável (3 meses)
**Primeira Meta:** Palestra técnica de 20 min

### 👤 Palestra Técnica, Sem Storytelling
**Trilha:** Intermediário (25-30h)
**Cronograma:** Moderado (1 mês)
**Primeira Meta:** Adicionar história pessoal (Módulo 4)

### 👤 Quero Domínio Completo
**Trilha:** Master (50-60h)
**Cronograma:** Intensivo ou Moderado
**Primeira Meta:** Evento real com 50+ pessoas

### ⏰ Evento Urgente (<2 semanas)
**Trilha:** Express (8-12h)
**Cronograma:** 1 semana intensiva
**IMPORTANTE:** Fazer trilha completa DEPOIS do evento!
