## 📖 Sobre Este Vault

Este vault Obsidian contém **todo o material necessário para facilitar o curso "Método Aplauda de Pé"** de Renner Silva.

Ele complementa a aplicação Next.js (plataforma do aluno) fornecendo:
- **MOCs (Maps of Content)** para cada módulo
- **Notas de conceitos** interconectadas com wikilinks
- **Templates** reutilizáveis para lições e exercícios
- **Navegação visual** via graph view
- **Sistema de tags** para organização

## 🚀 Como Usar Este Vault

### 1. Abrir no Obsidian

1. Baixe **[Obsidian](https://obsidian.md)** (gratuito)
2. Abra Obsidian
3. **"Open folder as vault"** → selecione `obsidian-vault/`
4. Pronto! Vault está configurado e pronto para uso.

### 2. Navegação Recomendada

#### 📍 Ponto de Partida
- Comece por **este README**
- Vá para **MOCs/** para visão geral de cada módulo

#### 🔍 Explorar Conceitos
- Use **graph view** (Ctrl/Cmd + G) para ver conexões
- Clique em wikilinks `[[Conceito]]` para navegar
- Tags: use `#modulo-1`, `#modulo-2`, etc para filtrar

#### 📝 Preparar Aula
1. Abra o MOC do módulo
2. Revise conceitos-chave
3. Use templates para criar materiais customizados
4. Consulte "Notas do Instrutor" em cada MOC

## 🔍 Recursos do Obsidian

### Graph View
- **Atalho:** Ctrl/Cmd + G
- Visualize conexões entre conceitos
- Identifique conceitos centrais (mais conexões)

### Backlinks
- Cada nota mostra onde é referenciada
- Útil para ver impacto de um conceito

### Tags
- `#modulo-1` a `#modulo-5` - Filtrar por módulo
- `#abertura`, `#promocao`, `#entrega`, `#historia`, `#finalizacao` - Por categoria

### Search
- **Atalho:** Ctrl/Cmd + Shift + F
- Busca em todo o vault
- Suporta regex

## 🎓 Dicas de Facilitação

### Antes da Aula
1. **Revise o MOC** do módulo
2. **Leia conceitos-chave** relacionados
3. **Prepare exemplos** adicionais (adapte ao contexto local)
4. **Configure ambiente** (projetor, materiais, etc)

### Durante a Aula
1. **Use MOC como guia** (estrutura clara)
2. **Navegue via wikilinks** para aprofundar conceitos
3. **Consulte "Notas do Instrutor"** para dicas específicas
4. **Registre insights** dos alunos como notas pessoais

### Depois da Aula
1. **Atualize notas** com insights da turma
2. **Adicione exemplos** que funcionaram bem
3. **Documente ajustes** para próxima turma

## 🛠️ Personalizações

### Adicionar Conceitos Novos
1. Use template de conceito (ou copie um existente)
2. Preencha todos os campos
3. Adicione wikilinks para relacionamentos
4. Adicione ao MOC apropriado

### Criar Lições Customizadas
1. Use `Templates/Template - Lição.md`
2. Preencha variáveis `{{VARIAVEL}}`
3. Salve em `Modulos/`
4. Linke do MOC

### Integração com Dados JSON
Os conceitos e estrutura deste vault espelham os dados JSON em:
- `../data/concepts.json`
- `../data/taxonomy.json`
- `../data/exercises.json`

**Sincronize** alterações entre vault e JSONs para manter coerência.

## 📖 Filosofia do Método

> **"Uma palestra não é sobre o que você conquistou. É sobre quem você se tornou."**

1. **Conexão Inicial** - Faça as pessoas gostarem de você
2. **Promoção do Conteúdo** - Crie desejo pelo conhecimento
3. **Entrega Estruturada** - Conteúdo em passos simples
4. **História de Essência** - Faça o público SENTIR
5. **Finalização Emocional** - Vídeo e aplauso de pé

**Resultado:** Aplauso de pé genuíno ❤️
