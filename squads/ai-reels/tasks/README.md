# AI Reels — Tasks

Este squad é um ambiente de produção de mídia (Remotion/React), não um squad tradicional de agentes.

As "tasks" são scripts de rendering e composições React, não arquivos markdown de tarefas. Este diretório existe para consistência estrutural do ecossistema.

Se no futuro o squad adotar tasks estruturadas (ex: render-batch, generate-template), elas serão definidas aqui.
