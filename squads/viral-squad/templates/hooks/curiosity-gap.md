# Curiosity Gap Hook Template

## Overview

**Retention:** 78%
**Best For:** Secrets, tips, discoveries, insider knowledge

## Formula

```
Tease valuable info + Create mystery around it
```

## Examples

### Fitness

```
VISUAL: Whispering to camera (intimate)
COPY: "Nutricionistas odeiam quando eu falo isso..."
AUDIO: ASMR whisper + suspense
```

### Business

```
VISUAL: Looking around nervously
COPY: "Ninguém fala sobre isso, mas..."
AUDIO: Soft tension build
```

### Lifestyle

```
VISUAL: Close-up secretive expression
COPY: "Depois de 10 anos eu descobri..."
AUDIO: Mystery music intro
```

## Visual Execution

1. **Frame 1 (0-0.5s):** Intimate/secretive setup
2. **Frame 2 (0.5-1.5s):** Lean in, whisper energy
3. **Frame 3 (1.5-3s):** Build anticipation

## Audio Execution

```
0.0-0.5s: Soft, intimate tone
0.5-2.0s: Build tension subtly
2.0-3.0s: Peak curiosity moment
```

## Copy Formulas

```
"Nobody tells you this about..."
"After [X years] I discovered..."
"The ONE thing that changed..."
"[Authority] don't want you to know..."
"Here's what they don't teach you..."
```

## Why It Works

Brain HATES unfinished loops.
Must know the answer to close the loop.
Creates forbidden knowledge feeling.
