# Pattern Interrupt Hook Template

## Overview

**Retention:** 82%
**Best For:** Controversial takes, myth-busting, unexpected reveals

## Formula

```
Unexpected Visual + Contradictory Statement
```

## Examples

### Fitness

```
VISUAL: Throwing "healthy" food in trash
COPY: "Você está comendo proteína ERRADO"
AUDIO: Scratch sound + dramatic music
```

### Business

```
VISUAL: Ripping up money/contract
COPY: "Esse 'investimento' está te deixando pobre"
AUDIO: Paper tear + tension build
```

### Education

```
VISUAL: Red X over common practice
COPY: "Pare de estudar assim (não funciona)"
AUDIO: Buzzer + whoosh
```

## Visual Execution

1. **Frame 1 (0-0.3s):** Unexpected action in progress
2. **Frame 2 (0.3-0.5s):** Text appears with impact
3. **Frame 3 (0.5-1s):** Reveal/reaction shot

## Audio Execution

```
0.0-0.2s: Silence or soft ambient
0.2-0.3s: IMPACT sound (scratch, whoosh, alert)
0.3-1.0s: Dramatic music begins
```

## Copy Formulas

```
"[Common belief] is actually [opposite]"
"You're doing [X] WRONG"
"Stop [common action] immediately"
"[Expert type] hate when I say this"
```

## Why It Works

Brain notices CHANGE. Scroll stops to process contradiction.
Cognitive dissonance creates curiosity gap.
