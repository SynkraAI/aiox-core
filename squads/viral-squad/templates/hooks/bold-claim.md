# Bold Claim Hook Template

## Overview

**Retention:** 80%
**Best For:** Educational content, tutorials, transformations

## Formula

```
Bold claim + Immediate proof promise
```

## Examples

### Fitness

```
VISUAL: Confident pose, direct camera
COPY: "Cardio não funciona (e eu posso provar)"
AUDIO: Confident music, authority tone
```

### Business

```
VISUAL: Numbers/results on screen
COPY: "Eu fiz R$100K em 30 dias com isso"
AUDIO: Success sound, energetic
```

### Education

```
VISUAL: Before/After split
COPY: "Isso mudou minha vida em 90 dias"
AUDIO: Transformation reveal music
```

## Visual Execution

1. **Frame 1 (0-0.5s):** Bold text + confident face
2. **Frame 2 (0.5-1.5s):** Proof teaser (results, data)
3. **Frame 3 (1.5-3s):** Promise of explanation

## Audio Execution

```
0.0-0.3s: Impact sound
0.3-1.5s: Confident, assertive delivery
1.5-3.0s: Build to explanation
```

## Copy Formulas

```
"[Bold statement] (here's why)"
"[Result] in [timeframe] with this"
"[Common belief] is a lie"
"I [achieved X] and you can too"
"This is the ONLY way to [result]"
```

## Why It Works

Challenges beliefs = emotional activation.
Proof promise = curiosity about evidence.
Confidence = authority and credibility.
