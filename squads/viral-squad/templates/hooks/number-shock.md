# Number Shock Hook Template

## Overview

**Retention:** 75%
**Best For:** Statistics, listicles, data-driven content

## Formula

```
Shocking statistic + Relevance to viewer
```

## Examples

### Fitness

```
VISUAL: Big number animation
COPY: "90% das pessoas fazem isso e NÃO emagrecem"
AUDIO: Alert sound + dramatic reveal
```

### Business

```
VISUAL: Money counter animation
COPY: "Você perde R$500/mês com isso"
AUDIO: Cash register + warning
```

### Health

```
VISUAL: Percentage bar filling
COPY: "73% das pessoas têm isso e não sabem"
AUDIO: Medical alert + concern
```

## Visual Execution

1. **Frame 1 (0-0.5s):** Number appears big
2. **Frame 2 (0.5-1.5s):** Context text
3. **Frame 3 (1.5-3s):** Personal relevance

## Number Animation Styles

```
COUNT UP: 0 → 90%
SLAM IN: Number impacts screen
REVEAL: Hidden → Shown
COMPARE: Your number vs Ideal
```

## Audio Execution

```
0.0-0.3s: Impact/alert sound
0.3-1.0s: Dramatic pause
1.0-3.0s: Explanation begins
```

## Copy Formulas

```
"[X%] of people do this and [bad result]"
"You lose [amount] every [time] by..."
"Only [small %] know this..."
"[Big number] [thing] and counting"
"[Shocking stat] (and it gets worse)"
```

## Why It Works

Numbers = credibility and specificity.
Statistics create urgency.
"Am I in that percentage?" = personal relevance.
