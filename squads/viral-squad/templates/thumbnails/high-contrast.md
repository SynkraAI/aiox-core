# High Contrast Thumbnail Template

## Overview

**CTR Boost:** +15-25%
**Best For:** Educational, dramatic, attention-grabbing

## Design Principles

### Color Contrast

```
BACKGROUND: Dark (#000000 or deep blue)
SUBJECT: Bright/Saturated
TEXT: White or Yellow
ACCENT: Gold sparingly (#C9B298)
```

### Academia Lendária Compliance

```
Background: 70% dark
Foreground: 22% light
Gold: MAX 8%
```

## Layout Structure

```
┌────────────────────┐
│                    │
│    [FACE/SUBJECT]  │
│      (40% frame)   │
│                    │
│  [BOLD TEXT]       │
│    (30% frame)     │
│                    │
│  [ACCENT ELEMENT]  │
│    (10% frame)     │
└────────────────────┘
```

## Text Rules

```
FONT: Inter SemiBold
SIZE: Readable at 100px height
MAX WORDS: 3-5
CONTRAST: White on dark
POSITION: Lower third or center
```

## Face Rules (If Using)

```
EXPRESSION: Exaggerated emotion
EYES: Looking at camera or text
SIZE: 40-60% of frame
LIGHTING: High contrast, dramatic
```

## Examples

### Tutorial Style

```
- Shocked face on left
- "VOCÊ ESTÁ ERRADO" in white
- Arrow pointing to face
- Dark gradient background
```

### Number Style

```
- Big number center (gold)
- Supporting text white
- Minimal face or none
- Pure black background
```

## Quality Checklist

- [ ] Readable at 100px
- [ ] Clear focal point
- [ ] High contrast colors
- [ ] 8% gold rule (if using gold)
- [ ] Emotion visible (if face)
- [ ] Brand consistent
