# Before/After Thumbnail Template

## Overview

**CTR Boost:** +20-30%
**Best For:** Transformations, comparisons, results

## Design Principles

### Split Layout

```
LEFT: Before state (desaturated, sad)
RIGHT: After state (vibrant, happy)
CENTER: Dividing line or arrow
```

### Visual Contrast

```
BEFORE: Muted colors, low energy
AFTER: Saturated, high energy
TRANSITION: Clear visual divide
```

## Layout Options

### Vertical Split

```
┌──────────┬──────────┐
│          │          │
│  BEFORE  │  AFTER   │
│          │          │
│  (muted) │ (vibrant)│
│          │          │
└──────────┴──────────┘
```

### Diagonal Split

```
┌────────────────────┐
│  BEFORE  ╲         │
│           ╲ AFTER  │
│            ╲       │
└────────────────────┘
```

### Arrow Transition

```
┌────────────────────┐
│  BEFORE  →  AFTER  │
│                    │
│  [RESULT TEXT]     │
└────────────────────┘
```

## Text Overlay

```
OPTION 1: Timeframe
"30 DIAS"

OPTION 2: Result
"-15KG"

OPTION 3: Minimal
Just the visual contrast
```

## Color Treatment

### Before Side

```
- Desaturate 30-50%
- Slightly darker
- Blue/gray tint
```

### After Side

```
- Full saturation
- Brighter
- Warm tint (gold accent OK)
```

## Examples

### Body Transformation

```
Before: Sad expression, loose clothes
After: Confident smile, fit appearance
Text: "90 DIAS" in center
```

### Skill Progress

```
Before: Messy/amateur result
After: Clean/professional result
Text: "Day 1 vs Day 365"
```

### Financial

```
Before: Empty wallet, stressed
After: Cash/success, confident
Text: "R$0 → R$100K"
```

## Quality Checklist

- [ ] Clear visual contrast
- [ ] Same person/subject both sides
- [ ] Readable transformation
- [ ] Emotion difference visible
- [ ] Text supports (not distracts)
- [ ] Works at small size
