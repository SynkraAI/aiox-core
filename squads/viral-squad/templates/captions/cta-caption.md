# 📝 Template: CTA Captions

## Overview

Call-to-Action captions that drive specific actions: saves, shares, comments, follows.

## CTA Psychology

```
Clear Action + Reason Why + Low Friction = High Conversion
```

## CTA by Goal

### 🔖 SAVE CTAs (Strongest Algorithm Signal)

```
Direct:
"Salva esse vídeo - você vai precisar depois 📌"
"Guarda pra consultar quando [situation] 💾"

Value-Based:
"Isso é checklist. Salva pra não esquecer ✅"
"Salva e usa como referência sempre 🔖"

Urgency:
"Salva antes que eu remova 👀"
"Guarda agora (depois não acha) 📌"
```

### 📤 SHARE CTAs (Viral Multiplier)

```
Tag Someone:
"Marca alguém que PRECISA ver isso 👥"
"Manda pro amigo que [specific situation] 📤"
"Tag aquela pessoa que sempre [behavior] 😂"

Forward:
"Compartilha nos stories (ajuda demais!) 🔄"
"Manda no grupo que vai dar treta 💬"
"Encaminha pra quem ainda não sabe 📨"
```

### 💬 COMMENT CTAs (Engagement Boost)

```
Simple Word:
"Comenta 'EU' se isso é você 🙋"
"Escreve 'QUERO' que te mando o link 👇"
"Deixa um 🔥 se concorda"

Opinion:
"Qual desses você prefere? A ou B? 💬"
"Concordas ou discordas? Me conta 🗣"
"O que você faria diferente? 👇"

Story:
"Conta sua experiência com [topic] 📝"
"Já passou por isso? Comenta aí 💭"
```

### 👤 FOLLOW CTAs

```
Value Promise:
"Segue pra mais [specific content type] 🔥"
"Me segue se quer [result] em [timeframe] ✅"
"Novo aqui? Posta [frequency] sobre [topic] 👋"

FOMO:
"Segue antes do próximo vídeo (amanhã) 🔔"
"80% do meu melhor conteúdo tá nos stories 📱"
```

### 🔗 LINK CTAs

```
"Link na bio pra [specific thing] 🔗"
"Clica no link da bio (é grátis) ✨"
"Detalhes completos no link da bio 📎"
```

## CTA Formulas por Formato

### Reels (Final Frame)

```
Visual: CTA text centered
Audio: Upbeat, energetic
Copy: "[Action verb] + [reason]"
Duration: 2-3 seconds
```

### Stories

```
Sticker: Poll, Quiz, or Question
Copy: Short, direct action
Add: Countdown or link sticker
```

### Feed Post

```
Caption CTA at end
First comment: Repeat CTA
Stories: Reshare with arrow pointing
```

## Multi-CTA Strategy

### Stack Order (Most to Least Important)

```
1. Primary CTA (what you REALLY want)
2. Secondary CTA (backup action)
3. Soft CTA (future relationship)

Example:
"Salva pra usar depois 📌
Comenta qual dica vai testar 👇
Segue pra mais conteúdo assim 🔥"
```

### Split Test Framework

```
Version A: Save-focused
Version B: Comment-focused
Version C: Share-focused

Track: Which drives best overall engagement?
```

## CTA Timing

| Video Length | CTA Position                |
| ------------ | --------------------------- |
| <15s         | Last 2-3s only              |
| 15-30s       | Soft @15s + Strong @end     |
| 30-60s       | Soft @20s + Strong @end     |
| >60s         | Multiple soft + Strong @end |

## Advanced Techniques

### The Double CTA

```
"Salva E comenta 'LISTA' que te mando o PDF completo 📌👇"
```

### The Conditional CTA

```
"Se chegou até aqui, você é diferente. Me segue. 🔥"
```

### The Reciprocity CTA

```
"Te dei [value]. Agora me ajuda: compartilha com 1 pessoa 🤝"
```

### The Social Proof CTA

```
"12.000 pessoas já salvaram. Você também? 📌"
```

## Don'ts ❌

- "Link na bio" sem especificar o quê
- CTAs vagos ("engaja aí")
- Muitos CTAs competindo
- CTA no início (soa desesperado)
- Pedir follow sem dar valor primeiro
