# 📝 Template: Hook Captions

## Overview

First line of caption = mini hook. Must stop the scroll AND complement the video.

## Formula

```
[HOOK LINE - stops scroll]
[CONTEXT - explains value]
[CTA - drives action]
```

## Hook Caption Formulas

### Curiosity Gap

```
"Isso vai mudar tudo que você sabe sobre [topic]"
"Descobri isso depois de [X years] e queria ter visto antes"
"Ninguém fala sobre isso mas [controversial truth]"
```

### Bold Claim

```
"[X] é mentira. Aqui está a verdade."
"Você está fazendo [X] errado (e não sabe)"
"[Number]% das pessoas não sabem disso"
```

### Direct Address

```
"Se você [condition], PRECISA ver isso"
"Isso é pra você que [struggle]"
"Para quem ainda acredita que [myth]..."
```

### Story Tease

```
"O dia que [dramatic event] mudou minha vida..."
"Eu estava [situation] quando percebi [insight]"
"Ninguém acreditou quando eu disse [claim]"
```

### Number Hook

```
"[Number] [things] que vão [result] em [timeframe]"
"Passei [time] testando. Aqui está o melhor."
"De [bad number] para [good number] em [time]"
```

## Structure Templates

### Short (Feed/Stories)

```
[1 line hook]
.
.
.
[CTA]
```

### Medium (Reels)

```
[Hook line]

[2-3 lines of context/value]

[CTA with emoji]
```

### Long (Carousel/IGTV)

```
[Strong hook]

[Paragraph of story/context]

[Bullet points of value]
• Point 1
• Point 2
• Point 3

[CTA]

[Hashtags - 3-5 max]
```

## CTA Formulas

### Save-Focused

```
"Salva pra consultar depois 📌"
"Guarda esse (vai precisar) 💾"
"Não perde esse conteúdo 🔖"
```

### Comment-Focused

```
"Comenta [word] que te mando 👇"
"Qual desses você vai testar? 💬"
"Concordas? Me conta aí 🗣"
```

### Share-Focused

```
"Marca alguém que precisa ver 👥"
"Manda pro amigo [type] 📤"
"Compartilha com quem [situation] 🔄"
```

### Follow-Focused

```
"Segue pra mais conteúdo assim 🔥"
"Novo aqui? Me segue! 👋"
"Ativa o sininho 🔔"
```

## Hashtag Strategy

```
Use 3-5 hashtags MAX (não 30!)

Formula:
- 1 niche específico (#fitnessmasculino)
- 1 mid-tier (#dicasdesaude)
- 1 broad relevante (#reels)
```

## Don'ts ❌

- Caption gigante antes do hook
- Hashtags no início
- CTAs genéricos ("link na bio")
- Emojis demais
- Repetir exatamente o que o vídeo diz
