# 📝 Template: Transformation Story

## Structure

```
[0-3s] HOOK
"90 days ago I was [bad state]. Watch what happened."
Visual: Split screen Before/After

[3-10s] SETUP
"I tried everything. Nothing worked. Until..."
Visual: Montage of struggles

[10-40s] JOURNEY
Day 1: [starting point]
Day 30: [first progress]
Day 60: [breakthrough]
Day 90: [final result]
Visual: Timeline with key moments

[40-55s] PAYOFF
"Here's exactly what I did: [3 key things]"
Visual: Checklist with results

[55-60s] CTA
"Which one will you start with? Comment below 👇"
Visual: Bold CTA with gold accent (8% rule)
```

## Variables

- Replace [bad state] com estado inicial
- Replace [key things] com 3 ações específicas
- Timing pode ajustar ±5s por seção

## Estimated Performance

- Retention @3s: 75-85%
- Completion rate: 45-55%
- Save rate: 8-12%
- Viral potential: HIGH
