# 📋 Template: Listicle (Number-Based)

## Structure

```
[0-5s] HOOK
"[Number] [things] that [result]. Number [X] is insane."
Visual: Bold number + preview of best item

[5-50s] RAPID FIRE LIST
Item 1: [2-5s each]
Item 2:
Item 3:
Item 4:
Item 5:
Visual: Fast cuts, kinetic typography

[50-55s] BONUS/SUMMARY
"Bonus tip: [extra value]"
Visual: Gold accent highlight

[55-60s] CTA
"Which one shocked you most? Drop a number 👇"
Visual: Numbers 1-5 on screen
```

## Numbers That Work

- 3 (quick, digestible)
- 5 (standard, trusted)
- 7 (curious, complete)
- 10 (comprehensive)

## Performance

- Fast pacing = High retention
- Easy to consume = High completion
- Number CTA = High comments
