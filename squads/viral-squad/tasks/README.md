# Viral Squad — Tasks

Este squad opera via workflows YAML (`workflows/`) e checklists (`checklists/`) em vez de tasks markdown individuais.

Os 7 agentes (viral-content, viral-strategy, viral-design, etc.) são orquestrados pelo squad.yaml, não por task files. Este diretório existe para consistência estrutural do ecossistema.

Se no futuro o squad adotar tasks individuais, elas serão definidas aqui.
