# Viral Frameworks Knowledge Base

## Hook Frameworks

### 1. Pattern Interrupt

**Formula:** Unexpected Visual + Contradictory Statement
**Retention:** 82%
**Examples:** Throwing away "healthy" food, Doing exercise "wrong"

### 2. Curiosity Gap

**Formula:** Tease valuable info + Create mystery
**Retention:** 78%
**Examples:** "Nobody tells you this...", "After 10 years I discovered..."

### 3. Shocking Statement

**Formula:** Bold claim + Immediate proof promise
**Retention:** 80%
**Examples:** "Cardio doesn't work", "You're wasting money on protein"

### 4. Question Hook

**Formula:** Relevant question they want answered
**Retention:** 75%
**Examples:** "Why can't you lose belly fat?", "What if I told you..."

### 5. Visual Spectacle

**Formula:** Impressive visual + Minimal text
**Retention:** 85%
**Examples:** Transformation split-screen, Impressive feat

## Script Frameworks

### PAS (Problem-Agitate-Solution)

```
[0-10s] PROBLEM: Identify pain point
[10-30s] AGITATE: Make it worse
[30-50s] SOLUTION: Present fix
[50-60s] CTA: Call to action
```

### BAB (Before-After-Bridge)

```
[0-10s] BEFORE: Current state
[10-30s] AFTER: Desired state
[30-50s] BRIDGE: How to get there
[50-60s] CTA
```

### AIDA

```
ATTENTION: Hook
INTEREST: Why care
DESIRE: What you get
ACTION: CTA
```
