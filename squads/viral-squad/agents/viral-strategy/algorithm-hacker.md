## Instagram Algorithm Deep Dive

### Ranking Signals (2025 Update)

**Tier 1: Strongest Signals** 🔥

```
1. SAVES (10x weight)
   - User saves = "this is valuable"
   - Algorithm: "Show this to more people"
   - How: Create save-worthy moments

2. SHARES (8x weight)
   - Sends to friends = viral signal
   - Algorithm: "People want others to see this"
   - How: Relatable/funny/shocking content

3. WATCH TIME (7x weight)
   - Completion rate + re-watches
   - Algorithm: "This is engaging"
   - How: Hook + retention optimization
```

**Tier 2: Important Signals** ⚡

```
4. COMMENTS (5x weight)
   - Quality > Quantity
   - Meaningful comments > emojis
   - How: Ask questions, create debate

5. LIKES (3x weight)
   - Still matters but less than before
   - Quick engagement signal
   - How: Emotional resonance

6. PROFILE VISITS (4x weight)
   - Interest in creator
   - Potential new follower
   - How: Compelling hook + value
```

**Tier 3: Secondary Signals** ✨

```
7. FOLLOWS (6x weight)
   - Best conversion signal
   - But happens to <5% of viewers
   - How: Consistent value + personality

8. STORY SHARES (4x weight)
   - Shares to own story
   - Shows strong connection
   - How: Make share-to-story worthy

9. DMs ABOUT POST (3x weight)
   - Deep engagement
   - Rare but powerful
   - How: Conversation-starting content
```

### The 3 Algorithm Stages

**Stage 1: Initial Test** (First 1 hour)

```
Algorithm shows to:
- ~5-10% of your followers
- Some non-followers (explore)

Measures:
- Engagement rate (first 30 min critical)
- Watch time
- Saves/shares

Decision:
IF engagement > threshold → Stage 2
IF engagement < threshold → STOP
```

**Stage 2: Performance Evaluation** (1-6 hours)

```
Algorithm analyzes:
- Completion rate
- Share rate
- Save rate
- Comment quality
- Profile visit rate

Compares to:
- Your previous posts
- Similar content performance
- Trending content benchmarks

Decision:
IF performance strong → Stage 3
IF performance weak → Slow down
```

**Stage 3: Distribution Decision** (6h - 7 days)

```
IF signals very strong:
- Push to Explore page
- Push to non-followers
- Viral distribution

IF signals good:
- Continue showing to followers
- Some explore exposure

IF signals weak:
- Stop distribution
- Archive in feed only
```

**First 30 minutes:**

## Hashtag Strategy (2025)

### Old Strategy (❌ Doesn't work anymore):

```
Use 30 hashtags
Mix of big and small
Hide in first comment
```

### New Strategy (✅ Works now):

```
Use 3-5 hashtags ONLY
Medium-sized hashtags (100K-5M posts)
In caption, not comments
Relevant to content (algorithm checks!)
```

### Why less is more?

**Instagram 2025 Update:**

- Algorithm prioritizes content quality over hashtag spam
- Too many hashtags = looks desperate
- Irrelevant hashtags = shadowban risk
- 3-5 relevant > 30 random

### Hashtag Research Process:

**Step 1: Find Medium Hashtags**

```
Sweet spot: 100K - 5M posts

Too small (<50K): Not enough reach
Too big (>10M): Too much competition

Example (Fitness):
✅ #fitnesstransformation (2.4M)
✅ #bodytransformation (1.8M)
❌ #fitness (500M+ - too big)
❌ #myfitnessjourney (30K - too small)
```

**Step 2: Check Top Posts**

```
Search hashtag
Look at Top 9 posts
If your content quality matches → use it
If Top posts are much better → skip
```

**Step 3: Monitor Performance**

```
Track which hashtags bring reach:
Instagram Insights → Reach → From Hashtags

Keep what works, drop what doesn't
Refresh every 2-4 weeks
```

```
This line decides if they click "more"

✅ Good: "This ONE change got me abs in 90 days 👇"
```

```
2-3 lines of value/story

Don't wall of text

```

## Shadowban: Detection & Recovery

### How to detect shadowban:

**Signs:**

1. Sudden drop in reach (>50%)
2. Not appearing in hashtag results
3. Only followers see your content
4. Insights show 0 reach from hashtags

**Check:**

```
1. Post with 1 hashtag
2. Ask friend (non-follower) to search hashtag
3. If your post doesn't appear in Recent → shadowbanned
```

### Common Causes:

1. **Too many hashtags** (>10)
2. **Irrelevant hashtags** (spam behavior)
3. **Bot-like behavior** (too many actions too fast)
4. **Reported content** (violations)
5. **Using banned hashtags** (yes, some are banned)

### Recovery Protocol:

**Step 1: Stop suspicious activity** (24-48h)

```
- Don't use hashtags
- Don't mass follow/unfollow
- Don't spam comments
- Post normal content
```

**Step 2: Review past content**

```
- Delete posts with banned hashtags
- Remove posts with violations
- Clean up irrelevant hashtags
```

**Step 3: Appeal** (if unfair)

```
Settings → Help → Report a Problem
Explain situation professionally
Wait 3-5 days for response
```

**Step 4: Rebuild trust** (2-4 weeks)

```
- Post high-quality content
- Use 3-5 relevant hashtags only
- Engage authentically
- Grow organically
```
