## Hook Frameworks

### 1. The Pattern Interrupt

**Formula:**

```
Unexpected Visual + Contradictory Statement
```

**Examples:**

- Throwing away "healthy" food
- Doing exercise "wrong"
- Reversing common wisdom

**Why it works:**
Brain notices CHANGE. Scroll stops to process.

---

### 2. The Curiosity Gap

**Formula:**

```
Tease valuable info + Create mystery around it
```

**Examples:**

- "Nobody tells you this about..."
- "After 10 years I discovered..."
- "The ONE thing that changed..."

**Why it works:**
Brain HATES unfinished loops. Must know answer.

---

### 3. The Shocking Statement

**Formula:**

```
Bold claim + Immediate proof promise
```

**Examples:**

- "Cardio doesn't work (here's why)"
- "You're wasting money on protein"
- "Everything you know is wrong"

**Why it works:**
Challenges beliefs = emotional activation.

---

### 4. The Question Hook

**Formula:**

```
Relevant question they want answered
```

**Examples:**

- "Why can't you lose belly fat?"
- "What if I told you..."
- "Ever wonder why..."

**Why it works:**
Brain automatically tries to answer = engagement.

---

### 5. The Visual Spectacle

**Formula:**

```
Impressive visual + Minimal text
```

**Examples:**

- Transformation split-screen
- Impressive feat (strength/skill)
- Beautiful cinematography

**Why it works:**
Visual cortex processes first = immediate impact.

---

### 6. The Controversy

**Formula:**

```
Polarizing opinion + "Fight me" energy
```

**Examples:**

- "Unpopular opinion:"
- "I'm gonna get canceled but..."
- "Hot take:"

**Why it works:**
Controversy = comments = algorithm boost.

---

### 7. The Relatability

**Formula:**

```
"You do this too" moment
```

**Examples:**

- "POV: You're trying to..."
- "We all do this..."
- "Tell me you're X without telling me"

**Why it works:**
Identity recognition = "this is me!"

---

### 8. The Tutorial Tease

**Formula:**

```
Show end result + Promise simple path
```

**Examples:**

- "How I got this body in 90 days"
- "The EASY way to..."
- "This ONE change..."

**Why it works:**
Transformation desire + simplicity promise.

---

### 9. The Mistake Reveal

**Formula:**

```
"Stop doing X" + Better alternative promise
```

**Examples:**

- "Stop wasting time on..."
- "You're doing this wrong..."
- "This is killing your progress"

**Why it works:**
Pain avoidance + rescue promise.

---

### 10. The Story Hook

**Formula:**

```
Start in middle of action/emotion
```

**Examples:**

- "I was 300 pounds when..."
- "The day everything changed..."
- "This almost ruined my life"

**Why it works:**
Human brain wired for stories = must know end.

## Visual Hook Techniques

### 1. Extreme Close-Up

Open with super close-up then pull back
**Effect:** Disorientation → Clarity (satisfying)

### 2. Reverse Slow-Mo

Start slow-mo, then normal speed
**Effect:** Cinematic feel = stops scroll

### 3. Split-Screen Contrast

Before/After, This/That side-by-side
**Effect:** Comparison = instant understanding

### 4. Fast Cuts (0.5s each)

Rapid montage of interesting shots
**Effect:** Energy = excitement

### 5. Unexpected Movement

Camera moves they don't expect
**Effect:** "Wait, what?" = engagement

### 6. Color Pop

Black & white → sudden color
**Effect:** Visual dopamine hit

### 7. Text Explosion

Text animates in dramatically
**Effect:** Readability + energy

## Copy Hook Formulas

### Formula 1: The Number

```
"[Number] [things] that [result]"
Example: "5 foods that KILL testosterone"
```

### Formula 2: The Timeline

```
"How to [result] in [timeframe]"
Example: "How to lose 10lbs in 30 days"
```

### Formula 3: The Contradiction

```
"[Common belief] is actually [opposite]"
Example: "Eating fat makes you LEAN"
```

### Formula 4: The Secret

```
"The [adjective] secret to [result]"
Example: "The stupid-simple secret to abs"
```

### Formula 5: The Personal

```
"How I [achieved] after [struggle]"
Example: "How I built muscle after 40"
```

### Formula 6: The Warning

```
"Stop [doing X] before [consequence]"
Example: "Stop cardio before you wreck your metabolism"
```

### Formula 7: The Question

```
"Why [problem]? (The answer will [emotion])"
Example: "Why can't you lose weight? (This will shock you)"
```

## Audio Hook Strategy

**First 0.5 seconds CRITICAL**

### Audio Types:

**1. Silence → Sudden Sound**

```
0.0-0.2s: Silence
0.2s: BANG/SCRATCH/ALERT
Effect: Attention grab
```

**2. Trending Audio**

```
Use first 3s of trending sound
People recognize = "oh I like this song" = watch
```

**3. Voice Impact**

```
Strong declarative statement
Confident, loud, clear
"STOP what you're doing"
```

**4. ASMR Whisper**

```
Intimate whisper
"Nobody talks about this but..."
Creates secret-sharing vibe
```

**5. Music Build**

```
Start low, build to drop at 3s
Tension → Release = satisfying
```

## Hook Testing Protocol

### Step 1: Create Variations

- Generate 10 hooks
- Rank by predicted performance
- Select top 3 for testing

### Step 2: Test Design

```
Video A: Hook 1 + Same content
Video B: Hook 2 + Same content
Video C: Hook 3 + Same content

Post 3-4 hours apart
Same hashtags, same time of day
```

### Step 3: Metrics to Track

- **3s Retention** (primary metric)
- CTR (thumbnail → video)
- Full video retention
- Engagement rate

### Step 4: Statistical Significance

```
Need ~10K views per variation
Winner = 10%+ better retention @3s
P-value < 0.05
```

### Step 5: Iterate

- Take winner
- Create 3 variations of winner
- Test again
- Compound improvements

## Hook Mistakes to Avoid

### ❌ Don't Do This:

**1. Slow Build**

```
"Hey guys, welcome back..."
❌ Nobody cares. Hook immediately.
```

**2. Selfie Static**

```
Just standing there talking
❌ Visual needs MOVEMENT
```

**3. Generic Opening**

```
"In this video I'm gonna show you..."
❌ Too meta. Just show it.
```

**4. Weak Promise**

```
"Some tips that might help"
❌ Confidence! "This WILL change everything"
```

**5. Cluttered Frame**

```
Too much going on visually
❌ Clear focal point needed
```

**6. Boring Text**

```
Small, slow-appearing text
❌ BOLD, FAST, CONTRAST
```
