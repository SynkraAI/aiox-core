## Trend Lifecycle Framework

### Estágios de Uma Trend

```
1. 🌱 BIRTH (0-3 dias)
   - Aparece em 1-2 accounts
   - <100K total views
   - Early adopter opportunity
   - AÇÃO: Experimental

2. 🚀 GROWTH (3-7 dias)
   - 5-20 accounts adaptam
   - 1M-50M views
   - Still early, high potential
   - AÇÃO: Adapt quickly

3. 🔥 PEAK (7-14 dias)
   - 50-500+ accounts
   - 100M-1B+ views
   - Saturação começando
   - AÇÃO: Add unique twist

4. 📉 DECLINE (14-21 dias)
   - Oversaturated
   - Diminishing returns
   - Late adopters
   - AÇÃO: Skip

5. ☠️ DEATH (21+ dias)
   - Considered "old"
   - Low engagement
   - Cringe factor
   - AÇÃO: Never touch
```

## Trend Signals (Como Identificar)

### Early Signals (Before it explodes)

**1. Rapid Comment Growth**

```
Video has 50K views but 5K comments = people care
High comment/view ratio = strong engagement
```

**2. Cross-Platform Jump**

```
TikTok → Instagram Reels = trend migrating
Usually 2-4 days delay between platforms
```

**3. Influencer Adoption**

```
Micro-influencers (10K-100K) testing = early
Macro-influencers (1M+) joining = trend peaking
```

**4. Audio Surge**

```
Audio usage growing 200%+ daily = strong signal
Same audio in multiple niches = universal appeal
```

**5. Variation Emergence**

```
People creating "my version" = trend has legs
Different niches adapting = long lifecycle
```

## Red Flags (Trends to Avoid)

### ❌ Skip These

**1. Dying Trends**

- Been around >2 weeks
- Declining engagement
- Feels "old"

**2. Niche-Specific Trends**

- Won't translate to your audience
- Requires specific context

**3. Controversial/Risky**

- Could harm brand
- Cancel-able content
- Platform violations

**4. Too Complex**

- Can't execute well quickly
- Requires resources you don't have

**5. Off-Brand**

- Doesn't fit your content
- Confuses audience
