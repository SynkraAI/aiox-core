## Reverse Engineering Process

### How to Reverse Engineer Viral Video

**Step 1: Collect Data**

```
- Views, likes, saves, comments
- Retention metrics (if available)
- Post time/day
- Caption structure
- Hashtags used
- Audio choice
```

**Step 2: Deconstruct Hook (0-3s)**

```
What stopped the scroll?
- Visual pattern interrupt?
- Bold text?
- Intriguing statement?
- Shocking image?

Rate hook: 1-10
```

**Step 3: Analyze Structure**

```
Map out video second-by-second:

0-3s: [Hook element]
3-10s: [Setup/value promise]
10-40s: [Value delivery method]
40-55s: [Payoff/climax]
55-60s: [CTA]

Identify retention techniques
```

**Step 4: Identify Viral Elements**

```
Check for:
✓ Relatability (is it "me"?)
✓ Shareability (would I share?)
✓ Save-worthy (useful info?)
✓ Emotional trigger (feel something?)
✓ Curiosity gap (want to know more?)
✓ Controversy (debate starter?)
```

**Step 5: Extract Formula**

```
[Hook type] + [Structure] + [Payoff] = Viral

Example:
Shocking stat + Listicle + Action steps = 2M views
```

## Gap Analysis

### Find Your Competitive Edge

**What They Do:**

- List all content types
- List all topics covered
- List all formats used

**What They Don't Do:**

- Missing content types?
- Untapped topics?
- Unused formats?

**Your Opportunity:**
Fill the gaps with quality content

### Gap Examples (Fitness):

```
Competitor A: All workout videos
Gap: No nutrition content
Your edge: Workout + meal prep combos

Competitor B: Only gym content
Gap: No home workouts
Your edge: Home workout alternatives

Competitor C: Advanced only
Gap: No beginner content
Your edge: Beginner-friendly approach
```

## Benchmarking

### Set Performance Targets

**Compare:**

```
You vs Competitor A:
- Views: You 50K | Them 200K (4x gap)
- Engagement: You 5% | Them 8% (1.6x gap)
- Growth: You +500/month | Them +2K/month

Target: Close gaps in 90 days
```

**Strategy:**

```
To match 200K views:
- Analyze their top 10 videos
- Extract common patterns
- Adapt for our audience
- Test and iterate
```

## Red Flags (Don't Copy These)

### Avoid:

❌ **Exact replication**

- Copyright issues
- No differentiation
- Looks desperate

❌ **Copying personality**

- Inauthentic
- Audience sees through it

❌ **Copying ALL trends**

- No unique voice
- Inconsistent brand

✅ **Do instead:**

- Inspired by, not copied from
- Add your unique twist
- Stay authentic
- Build own brand
