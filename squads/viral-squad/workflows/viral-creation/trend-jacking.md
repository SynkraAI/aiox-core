# ⚡ Trend Jacking Sprint - Workflow Rápido

## Objetivo

Pegar carona em trend viral com velocidade máxima (4-8h)

## Agentes: @trend-hunter (líder), @viral, @script-architect, @remotion-architect, @render-master

## Workflow com Debate

### FASE 1: CAPTURA (30 min)

```
@trend-hunter: *hunt-trends [NOW]
DEBATE:
- Trend lifecycle? (nascendo/pico)
- Window? (dias restantes)
- Fit? (alinha com marca)
CONSENSO: Trend selecionada
```

### FASE 2: ADAPTAÇÃO (2h)

```
@viral: *viral-strategy [trend adaptation]
@script-architect: *viral-script [quick version]
DEBATE:
- Twist único? (diferenciação)
- Mantém essência do trend?
CONSENSO: Script adaptado
```

### FASE 3: PRODUÇÃO (2-4h)

```
@remotion-architect: Template pre-existente
@render-master: Fast render (CRF 28)
DEBATE: Quality vs Speed trade-off
CONSENSO: Render iniciado
```

### FASE 4: LANÇAMENTO (1h)

```
Post imediatamente
Share to Story
DM engaged followers
```

**Tempo Total: 4-8 horas**
**Output:** Vídeo surfando trend em tempo real
