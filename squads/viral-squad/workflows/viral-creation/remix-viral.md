# 🔄 Remix Viral Hit - Workflow de Remixagem

## Objetivo

Remixar vídeo viral de competidor com twist único (2-3 dias)

## Agentes: @research-ninja (líder), @viral, @script-architect, @visual-impact, @remotion-architect

## Workflow com Debate

### FASE 1: REVERSE ENGINEERING (4h)

```
@research-ninja: *reverse-engineer [viral video]
Output: Elementos virais identificados
DEBATE:
- O que fez viralizar?
- Qual elemento podemos melhorar?
CONSENSO: Core viral element extraído
```

### FASE 2: DIFERENCIAÇÃO (1 dia)

```
@viral: Identificar twist único
@script-architect: Remixar roteiro (70% similar, 30% único)
DEBATE:
- Twist suficiente para diferenciar?
- Mantém elementos virais?
CONSENSO: Remix aprovado
```

### FASE 3: PRODUÇÃO (1 dia)

```
@visual-impact: Better execution
@remotion-architect: Implementar
DEBATE: Qualidade superior ao original?
CONSENSO: Produção finalizada
```

**Tempo Total: 2-3 dias**
**Output:** Remix com twist único
