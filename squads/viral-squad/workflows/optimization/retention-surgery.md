# ⚕️ Retention Surgery - Fix Baixa Retenção

## Objetivo

Corrigir vídeo com baixa retenção (<40%) (2-3 dias)

## Agentes: @retention-architect (líder), @hook-master, @script-architect, @motion-master

## Workflow com Debate

### FASE 1: DIAGNÓSTICO (2h)

```
@retention-analyst: *retention-deep-dive
@retention-architect: *drop-off-analysis
DEBATE:
- Drop-offs críticos identificados
- Causa raiz de cada drop
CONSENSO: Problemas mapeados
```

### FASE 2: PRESCRIÇÃO (4h)

```
Para cada problema:
- Hook issue → @hook-master *create-hooks
- Pacing issue → @retention-architect *optimize-pacing
- Value issue → @script-architect restructure

DEBATE: Melhor solução para cada?
CONSENSO: Fixes aprovados
```

### FASE 3: CIRURGIA (1-2 dias)

```
Implementar fixes
Re-render
A/B test vs original
```

### FASE 4: VALIDAÇÃO (1 dia)

```
@metrics-guru: Compare retention graphs
DEBATE: Melhoria suficiente? (target +20%)
CONSENSO: Surgery bem-sucedida ou iterar
```

**Tempo Total: 2-3 dias**
**Output:** Vídeo com +20-30% retenção
