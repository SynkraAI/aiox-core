## Fase 2: Engagement Mechanics Design (3-4 horas)

### 2.1 In-Video Engagement Triggers

**@engagement-engineer + @retention-architect:**

```markdown
ADD TO VIDEO:

1. CURIOSITY LOOPS (every 15s)
   - "Wait until you see what happens next"
   - Creates anticipation = longer watch

2. PATTERN INTERRUPTS (3-5 per video)
   - Visual shock
   - Audio spike
   - Unexpected reveal

3. MICRO-COMMITMENTS
   - "Drop 🔥 if you agree"
   - "Comment X if you want part 2"
   - "Save this now, thank me later"

4. CONTROVERSIAL TAKES
   - "Unpopular opinion: [X]"
   - "Everyone says [Y] but [Z]"
   - Sparks debate

5. QUESTION SETUPS
   - "Which would you choose?"
   - "What would you do?"
   - "Have you experienced this?"
```

## Fase 3: Comment Farming Strategy (1-2 horas)

### 3.1 Reply Strategy

**@engagement-engineer cria playbook:**

```markdown
REPLY PROTOCOL (First 2 hours):

1. REPLY ALL COMMENTS <5min
   - Personalized responses
   - Ask follow-up question
   - Create dialogue

2. HIGHLIGHT TOP COMMENTS
   - Pin user success stories
   - Feature best questions
   - Acknowledge controversy

3. SEED DISCUSSION
   - Reply com new perspectives
   - Tag outros commenters
   - Create mini-threads

4. REWARD ENGAGEMENT
   - "DM me for [bonus]" (top commenters)
   - "Great point! 🔥" (validation)
   - "This deserves its own video"

GOAL: Turn comments section into community discussion
```

### 3.2 Question Sequences

**@copy-wizard cria question ladder:**

```markdown
QUESTION LADDER (nos replies):

Level 1 (Easy):
"Qual você tentou primeiro?"
(Gets initial comment)

Level 2 (Medium):
"O que funcionou melhor no seu caso?"
(Gets deeper engagement)

Level 3 (Hard):
"Se você pudesse fazer diferente, o que mudaria?"
(Gets story/insight)

Level 4 (Personal):
"Posso usar seu case num próximo vídeo?"
(Gets permission + future engagement)

Each level aumenta investment do commenter
```

### 3.3 Controversy Management

**@viral + @engagement-engineer:**

```markdown
HANDLING DISAGREEMENT (productive controversy):

DO's ✅:

- "Interessante! Me explica mais sobre [X]"
- "Bom ponto. Já considerou [Y]?"
- "Dados mostram [Z], mas sua experiência é válida"
- "Let's discuss: [question]"

DON'Ts ❌:

- "Você está errado"
- "Leia de novo"
- "Você não entendeu"
- Defensive replies

GOAL: Controversy = engagement, não alienation
```

## Pro Tips

### Do's ✅

- Reply ALL comments first 2h
- Use multiple engagement mechanics
- Make commenting EASY (polls, emojis)
- Reward top commenters (DMs, shoutouts)
- Create conversation, not monologue

### Don'ts ❌

- Não peça "like" (doesn't matter)
- Não ignore comments (kills momentum)
- Não seja defensive (embrace controversy)
- Não seja genérico (specific > broad)
- Não esqueça save CTAs (highest weight)
