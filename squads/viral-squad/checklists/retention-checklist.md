# Retention Optimization Checklist

## Critical Retention Points

### 0-1 Second (Scroll Stop)

- [ ] **Visual pattern interrupt** - Algo inesperado na tela
- [ ] **Movement** - Não estático, algo se movendo
- [ ] **High contrast** - Cores que se destacam no feed
- [ ] **Face/Eyes** - Rostos atraem atenção naturalmente
- [ ] **Text hook visible** - Se usar texto, legível instantaneamente

### 1-3 Seconds (Hook Window)

- [ ] **Curiosity gap aberto** - "O que vai acontecer?"
- [ ] **Value promise** - Razão clara para continuar
- [ ] **Audio hook** - Som que prende (música, voz, SFX)
- [ ] **Tension initiated** - Conflito ou problema apresentado
- [ ] **NO intro genérico** - Nada de "oi gente, tudo bem?"

### 3-7 Seconds (Commitment Point)

- [ ] **Promise reinforced** - Confirma que vale a pena
- [ ] **Progress indicator** - Sensação de que está avançando
- [ ] **New information** - Algo que não sabia antes
- [ ] **Emotional connection** - Relatabilidade ou empatia
- [ ] **Pacing maintained** - Não desacelera

---

## Retention Killers (Avoid)

### Visual Killers

- [ ] **NO tela preta/estática** - Sempre algo visual
- [ ] **NO talking head sem movimento** - Adicionar B-roll, text, zoom
- [ ] **NO texto pequeno** - Mobile-first, text grande
- [ ] **NO cluttered frame** - Foco claro em um elemento
- [ ] **NO cores apagadas** - Contraste e saturação adequados

### Audio Killers

- [ ] **NO silêncio prolongado** - Sempre som ambiente ou música
- [ ] **NO áudio baixo** - Volume consistente e audível
- [ ] **NO eco/ruído** - Qualidade de áudio profissional
- [ ] **NO música irritante** - Combina com mood do conteúdo
- [ ] **NO voice monotone** - Variação de tom e energia

### Pacing Killers

- [ ] **NO dead air** - Cortar pausas desnecessárias
- [ ] **NO repetição** - Cada segundo é novo valor
- [ ] **NO filler words** - "Tipo", "né", "então" em excesso
- [ ] **NO setup longo** - Ir direto ao ponto
- [ ] **NO final arrastado** - Terminar no pico, não depois

---

## Tension Curve Optimization

### Structure Check

- [ ] **Tension at start** - Abre com energia alta
- [ ] **Build throughout** - Vai aumentando
- [ ] **Mini-peaks** - Pequenos payoffs durante
- [ ] **Main climax** - Ponto mais alto próximo ao final
- [ ] **Satisfying end** - Resolução que justifica a jornada

### Pacing Pattern

```
Good: [HIGH] → [build] → [peak] → [build] → [CLIMAX] → [resolution]
Bad:  [low] → [low] → [low] → [suddenly high] → [drop]
```

## Drop Point Analysis

### If Dropping at 0-3s

- [ ] Hook mais forte - Pattern interrupt mais impactante
- [ ] Visual mais chamativo - Cores, movimento, face
- [ ] Texto mais bold - Claim mais provocativo
- [ ] Audio impact - Som nos primeiros 0.5s

### If Dropping at 3-10s

- [ ] Value mais claro - O que vão ganhar?
- [ ] Pacing mais rápido - Cuts mais frequentes
- [ ] Promise reforçada - Lembrar por que assistir
- [ ] Curiosity ampliada - Mais mistério/antecipação

### If Dropping at 10-30s

- [ ] Conteúdo mais denso - Mais valor por segundo
- [ ] Variação visual - Mudar enquadramento/cena
- [ ] B-roll interessante - Não só talking head
- [ ] Progress cues - "Em primeiro lugar...", "Número 1..."

### If Dropping Before End

- [ ] Payoff mais forte - Final vale a espera?
- [ ] Tease final - "O último é o mais importante"
- [ ] Loop setup - Fim conecta com início
- [ ] CTA compelling - Razão para ficar até o fim
