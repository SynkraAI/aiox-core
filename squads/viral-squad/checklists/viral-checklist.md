# ✅ Viral Video Checklist

## Pre-Production

- [ ] Hook retém >75% @3s (tested)
- [ ] Script tem tension curve clara
- [ ] Visual para o scroll (pattern interrupt)
- [ ] CTA claro e específico
- [ ] Save-worthy moment incluído
- [ ] Share trigger inserido

## Production

- [ ] 8% gold rule respeitado
- [ ] High contrast (black/white/gold)
- [ ] Mobile-optimized (1080x1920)
- [ ] 60fps mantido
- [ ] Audio trending ou viral-worthy
- [ ] Transitions suaves

## Pre-Publish

- [ ] Caption hook forte (<125 chars)
- [ ] 3-5 hashtags relevantes escolhidos
- [ ] Thumbnail high-contrast approved
- [ ] Timing otimizado (audience insights)
- [ ] Post-publish plan ready (30min)

## Post-Publish (First 2h)

- [ ] Reply ALL comments <5min
- [ ] Share to Story c/ poll/question
- [ ] DM 20 engaged followers
- [ ] Monitor engagement velocity
- [ ] Adjust if needed

## Metrics Check (24h)

- [ ] Retention @3s >75%?
- [ ] Save rate >8%?
- [ ] Share rate >3%?
- [ ] Profile visits >15%?
- [ ] If NO to 2+ → Run optimization
