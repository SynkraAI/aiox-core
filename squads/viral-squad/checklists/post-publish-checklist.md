# Post-Publish Actions Checklist

## Immediate (0-30 Minutes) - CRITICAL

### Engagement Boost

- [ ] **Reply to EVERY comment** - Quantidade e qualidade
- [ ] **Ask follow-up questions** - Gera mais replies
- [ ] **Heart all comments** - Mostra que está ativo
- [ ] **Pin best comment** - Destaque um comentário top

### Cross-Promotion

- [ ] **Share to Stories** - Com engagement sticker (poll/question)
- [ ] **Add "New Post" sticker** - Direciona para o Reel
- [ ] **DM 10-20 engaged followers** - "Acabei de postar algo que acho que você vai curtir"

---

## First 2 Hours

### Monitoring

- [ ] **Check initial metrics** - Views, likes, comments
- [ ] **Watch retention curve** - Onde está dropando?
- [ ] **Monitor comment sentiment** - Positivo/negativo?
- [ ] **Track saves/shares** - Sinais mais importantes

### Continued Engagement

- [ ] **Second wave of replies** - Não responder tudo de uma vez
- [ ] **Thank supporters** - Reconhecer quem engajou
- [ ] **Engage thoughtfully** - Respostas substantivas

---

## First 24 Hours

### Metrics Analysis

- [ ] **Views vs followers** - View rate (target: 100%+)
- [ ] **Retention @3s** - Target: 75%+
- [ ] **Save rate** - Target: 5%+ (viral: 8%+)
- [ ] **Share rate** - Target: 1.5%+ (viral: 3%+)
- [ ] **Profile visits** - Convertendo curiosidade?
- [ ] **Follows** - Crescimento de seguidores

### Performance-Based Actions

#### If VIRAL (>10x normal views)

- [ ] Não fazer nada drástico - Deixar algoritmo trabalhar
- [ ] Planejar follow-up content - Capitalizar momentum

#### If GOOD (2-5x normal views)

- [ ] Analisar o que funcionou - Hook? Topic? Timing?
- [ ] Replicar elementos - No próximo vídeo

#### If POOR (<1x normal views)

- [ ] Não deletar - Pode pegar depois
- [ ] Analisar profundamente - O que deu errado?

---

## 48-72 Hours

### Documentation

- [ ] **Update metrics tracker** - Planilha de performance
- [ ] **Note learnings** - O que funcionou/não funcionou
- [ ] **Plan next content** - Baseado em insights
