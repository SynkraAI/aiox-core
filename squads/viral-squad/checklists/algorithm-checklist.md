# Algorithm Optimization Checklist

## Pre-Publish Algorithm Signals

### Content Quality Signals

- [ ] **Hook strength** - Pattern interrupt nos primeiros 0.5s
- [ ] **3s retention optimized** - Curva de tensão validada
- [ ] **Full watch incentive** - Payoff no final que justifica assistir tudo
- [ ] **Re-watch value** - Elementos que fazem querer ver de novo
- [ ] **Audio synced** - Som alinhado com visual para dopamine hits

### Save Triggers (10x weight - MOST IMPORTANT)

- [ ] **Actionable content** - Tutorial, how-to, tips práticos
- [ ] **Reference value** - Conteúdo para consultar depois
- [ ] **Checklist/List format** - Fácil de salvar e revisar
- [ ] **Save CTA** - "Salva pra não esquecer" ou similar
- [ ] **Visual save moment** - Tela clara que vale screenshot

### Share Triggers (8x weight)

- [ ] **Relatable moment** - "Isso é muito eu" / tag a friend
- [ ] **Controversy/Hot take** - Opinião polarizante mas não ofensiva
- [ ] **Useful for others** - "Minha amiga PRECISA ver isso"
- [ ] **Share CTA** - "Manda pro amigo que precisa"
- [ ] **Emotional peak** - Momento que gera reação de compartilhar

### Watch Time Signals (7x weight)

- [ ] **No dead moments** - Cada segundo tem valor
- [ ] **Pacing otimizado** - Ritmo que mantém atenção
- [ ] **Tension curve** - Build-up → climax → resolution
- [ ] **Loop potential** - Fim conecta com início
- [ ] **Optimal length** - Duração ideal para o conteúdo (não esticar)

### Follow Triggers (6x weight)

- [ ] **Series tease** - "Parte 2 amanhã" ou similar
- [ ] **Expertise demonstration** - Mostra que você é autoridade
- [ ] **Personality peek** - Algo único sobre você/marca
- [ ] **Follow CTA** - Motivo claro para seguir
- [ ] **Content promise** - O que vão ganhar seguindo

### Comment Triggers (5x weight)

- [ ] **Question CTA** - Pergunta específica no final
- [ ] **Opinion request** - "Concorda ou discorda?"
- [ ] **Poll/Choice** - "Time A ou Time B?"
- [ ] **Controversy bait** - Tema que gera debate
- [ ] **Fill in blank** - "O melhor **\_** é..."

### Profile Visit Signals (4x weight)

- [ ] **Bio mention** - "Link na bio" quando relevante
- [ ] **Curiosity about creator** - "Quem é essa pessoa?"
- [ ] **More content promise** - "Tenho mais sobre isso no perfil"

## Caption Optimization

### Caption Structure

- [ ] **Hook na primeira linha** - Antes do "mais..."
- [ ] **Value statement** - Por que assistir
- [ ] **Keywords naturais** - Termos que a audiência busca
- [ ] **CTA claro** - Uma ação específica
- [ ] **Hashtags otimizados** - 3-5 relevantes (não 30)

### Hashtag Strategy

- [ ] **Niche hashtag** - 1-2 específicos do nicho (10K-500K posts)
- [ ] **Medium hashtag** - 1-2 médios (500K-5M posts)
- [ ] **Broad hashtag** - 1 amplo para discovery (5M+ posts)
- [ ] **Trending hashtag** - Se relevante e atual
- [ ] **NO banned hashtags** - Verificar se algum está shadowbanned

## Post-Publish Algorithm Boost

### First 30 Minutes (CRITICAL)

- [ ] **Responder TODOS os comentários** - Engagement inicial
- [ ] **Comentários longos** - Respostas substantivas, não só emoji
- [ ] **Fazer perguntas de volta** - Gera mais replies
- [ ] **Pin best comment** - Destaca comentário valioso

### First 2 Hours

- [ ] **Share to Stories** - Com poll/question sticker
- [ ] **DM para engaged followers** - "Acabei de postar, acho que você vai curtir"
- [ ] **Cross-platform share** - TikTok, Twitter se relevante

### First 24 Hours

- [ ] **Monitorar métricas** - Retention curve, engagement rate
- [ ] **Reply em waves** - Não tudo de uma vez
- [ ] **Engage com outros posts** - Reciprocidade
- [ ] **Se >100K views** - Considerar boost pago

## Red Flags (Algorithm Killers)

### Content Red Flags

- [ ] **NO watermarks** - TikTok logo = death sentence
- [ ] **NO low resolution** - Mínimo 1080p
- [ ] **NO copyrighted music** - Use trending sounds ou royalty-free
- [ ] **NO sensitive topics** - Política, violência, etc sem cuidado
- [ ] **NO clickbait sem delivery** - Promessa tem que ser cumprida

### Technical Red Flags

- [ ] **NO aspect ratio errado** - 9:16 para Reels
- [ ] **NO áudio dessincronizado** - Lip sync perfeito
- [ ] **NO framedrops** - Smooth 30/60fps
- [ ] **NO compression artifacts** - Export em qualidade alta

### Engagement Red Flags

- [ ] **NO comprar engajamento** - Bots = shadowban
- [ ] **NO engagement pods** - Instagram detecta
- [ ] **NO spam comments** - Qualidade > quantidade
- [ ] **NO deletar e repostar** - Prejudica reach
