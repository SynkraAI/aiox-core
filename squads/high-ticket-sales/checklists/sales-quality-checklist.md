# Sales Quality Checklist

Quality gate for reviewing ANY sales asset (script, pitch, funnel, email sequence). Multi-framework audit.

**Purpose:** Ensure sales assets follow proven frameworks (SPIN, Challenger, Frame Control, Objection Handling, Value Equation).

---

## SPIN Discovery (Rackham)

### Situation Questions
- [ ] Opens with rapport + situation questions (understand current state)
- [ ] Avoids interrogation (max 2-3 situation questions)
- [ ] Establishes context before diving into problems

### Problem Questions
- [ ] Identifies the specific pain/problem
- [ ] Asks about impact of the problem (emotional + financial)
- [ ] Multiple problem questions to uncover depth

### Implication Questions
- [ ] Explores consequences if problem is NOT solved
- [ ] Amplifies pain (what happens 6 months from now?)
- [ ] Connects problem to bigger stakes (business, family, identity)

### Need-Payoff Questions
- [ ] Asks prospect to articulate the value of solving the problem
- [ ] Prospect says "yes" to the solution before it's presented
- [ ] Builds desire and commitment before pitch

## Frame Control (Klaff)

- [ ] Seller/agent controls the frame from the start
- [ ] Status reframe: Seller is the prize, not the prospect
- [ ] Time constraints set by seller (not prospect)
- [ ] Disqualification present ("This isn't for everyone")
- [ ] Avoids neediness or chasing energy
- [ ] Uses pattern interrupts to break prospect's autopilot

## Objection Handling (Voss + Ziglar)

### Top 5 Objections Addressed
- [ ] **Price:** "It's too expensive" → Reframe to value/ROI
- [ ] **Time:** "I don't have time" → Reframe to cost of inaction
- [ ] **Spouse:** "I need to talk to my partner" → Pre-handled or welcomed
- [ ] **Skepticism:** "I've tried this before" → Differentiation/new mechanism
- [ ] **Timing:** "Not right now" → Urgency/scarcity reinforced

### Objection Handling Technique (Voss)
- [ ] Label the objection: "It sounds like price is a concern..."
- [ ] Accusation audit: Pre-empt objections before they arise
- [ ] Mirroring: Repeat last 3 words to get prospect to elaborate
- [ ] Calibrated questions: "What about this concerns you?"
