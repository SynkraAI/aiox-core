## Checklist

### Step 1: Design Lead Magnet ($0 — Entry Point)

**Purpose:** Capture attention, build list, qualify prospects

- [ ] Identify the "one thing" they want most: Quick win, template, checklist, training
- [ ] Choose lead magnet format:
  - [ ] PDF guide/checklist
  - [ ] Video training (15-30 min)
  - [ ] Webinar (live or evergreen)
  - [ ] Quiz/assessment
  - [ ] Template/swipe file
  - [ ] Free trial (software)
- [ ] Name the lead magnet: "[Outcome] in [Timeframe] without [Pain]"
  - Example: "Generate Your First $10K Month Without Paid Ads"
- [ ] Deliver immediate value: Does this create a quick win?
- [ ] Set up the next offer: Does this naturally lead to Tripwire?
- [ ] Create opt-in page: Headline + 3 bullets + form
- [ ] Design thank-you page: Deliver lead magnet + introduce Tripwire (soft pitch)

**Lead Magnet Success Criteria:**
- Solves one specific problem
- Deliverable in <30 minutes
- Demonstrates expertise
- Creates desire for more

### Step 2: Design Tripwire ($7-$97 — Low-Commitment Offer)

**Purpose:** Convert freebie seekers to buyers, cover ad costs

- [ ] Choose Tripwire format:
  - [ ] Mini-course (3-5 videos)
  - [ ] Template pack
  - [ ] Book/guide (extended version)
  - [ ] 1-hour consultation
  - [ ] Software trial (14-30 days)
- [ ] Price at impulse level: $7-$97 (must be no-brainer)
- [ ] Create irresistible offer: Stack value 10x+ price
- [ ] Add urgency: Discount expires in 30 min? Limited to first-time buyers?
- [ ] Position as "bridge": "You got the overview (lead magnet), now get the system (Tripwire)"
- [ ] Design order bump: Add $17-$47 bonus at checkout
- [ ] Design sales page: VSL or sales letter format
- [ ] Plan delivery: Immediate access after payment

**Tripwire Success Criteria:**
- Conversion rate: 5-20% of lead magnet buyers
- Perceived value: 10x+ actual price
- Ascension path: Naturally leads to Core Offer

### Step 3: Design Core Offer ($297-$997 — Main Product)

**Purpose:** Deliver full transformation, generate profit

- [ ] Review offer architecture (from `build-offer.md`)
- [ ] Position in value ladder: "You got the framework (Tripwire), now get the implementation (Core)"
- [ ] Choose sales mechanism:
  - [ ] Webinar (live or evergreen)
  - [ ] VSL (video sales letter)
  - [ ] Sales call (if high-touch)
  - [ ] Email sequence → sales page
- [ ] Create pitch (from `create-pitch.md` if applicable)
- [ ] Design application process: Who qualifies? Automatic or manual approval?
- [ ] Set conversion goal: 10-30% of Tripwire buyers → Core
- [ ] Plan onboarding: First 30 days roadmap
- [ ] Identify upsell opportunity: What's the natural next step?

**Core Offer Success Criteria:**
- Delivers full promised transformation
- High perceived value (student ROI)
- Clear ascension path to High-Ticket

### Step 4: Design High-Ticket ($2K-$25K+ — Premium Offer)

**Purpose:** Serve best clients, maximize LTV

- [ ] Define High-Ticket offer:
  - [ ] 1-on-1 coaching/consulting
  - [ ] Done-for-you service
  - [ ] Mastermind/group program
  - [ ] VIP day/intensive
  - [ ] Implementation partnership
- [ ] Position as exclusive: Limited spots, application-only, invite-only
- [ ] Create qualification criteria: Who is a good fit? Revenue/commitment level?
- [ ] Design sales process:
  - [ ] Application form (filter unqualified)
  - [ ] Strategy call (from `create-sales-script.md`)
  - [ ] Enrollment call (if qualified)
- [ ] Set pricing: Based on value delivered, not hours worked
- [ ] Plan delivery: 90-day program? 6-month engagement? Ongoing?
- [ ] Target conversion: 10-30% of Core buyers → High-Ticket

**High-Ticket Success Criteria:**
- Premium positioning (not for everyone)
- High client satisfaction (referrals)
- Profitable at scale (high margin)

### Step 5: Build Email Nurture Sequence

**Purpose:** Build trust, ascend buyers through value ladder

**Sequence 1: Lead Magnet → Tripwire (5-7 days)**
- [ ] Email 1 (Immediate): Deliver lead magnet, introduce yourself
- [ ] Email 2 (Day 1): Share story/case study, soft pitch Tripwire
- [ ] Email 3 (Day 2): Teach one thing, CTA to Tripwire
- [ ] Email 4 (Day 3): Break false belief, Tripwire offer
- [ ] Email 5 (Day 5): Testimonial/social proof, last chance Tripwire
- [ ] Email 6 (Day 7): Final pitch, urgency/scarcity
- [ ] Email 7+ (Ongoing): Weekly value emails (nurture for Core offer)

**Sequence 2: Tripwire → Core Offer (7-14 days)**
- [ ] Email 1: Congratulate on purchase, set expectations
- [ ] Email 2: Quick win, case study, introduce Core
- [ ] Email 3: Teach belief break #1
- [ ] Email 4: Teach belief break #2
- [ ] Email 5: Teach belief break #3
- [ ] Email 6: Webinar invite / sales call booking
- [ ] Email 7+: Webinar reminder, replay, urgency

**Sequence 3: Core → High-Ticket (30-90 days)**
- [ ] Identify top performers in Core (engagement, results)
- [ ] Email 1: "I've been watching your progress..."
- [ ] Email 2: "I have something I think you'd be perfect for..."
- [ ] Email 3: Application / strategy call invitation
- [ ] Follow-up: Personal outreach via DM/phone

### Step 6: Build Follow-Up System (Recovery)

**Purpose:** Convert "not yet" to "yes"

**Non-Buyers (Didn't Buy Tripwire):**
- [ ] Day 7: "What held you back?" survey
- [ ] Day 14: Different angle (new case study)
- [ ] Day 30: Re-pitch with new urgency/bonus
- [ ] Ongoing: Retarget with ads, weekly value emails

**Tripwire Buyers (Didn't Buy Core):**
- [ ] Week 2: Check-in, "How's it going?"
- [ ] Week 4: Success story, "This could be you"
- [ ] Week 8: Re-pitch Core with new mechanism (webinar → VSL)
- [ ] Ongoing: Monthly touchpoints, flash sales

**Core Buyers (Didn't Buy High-Ticket):**
- [ ] Month 2: Check-in call, "What's working? What's not?"
- [ ] Month 3: Introduce High-Ticket as "next level"
- [ ] Month 6: Re-qualify, personal invite
- [ ] Ongoing: Quarterly touchpoints, exclusive offers

### Step 7: Map Funnel Flow & Metrics

**Draw the Funnel:**
```
Traffic Source (1000 visitors)
    ↓
Lead Magnet Opt-In (30% = 300 leads)
    ↓
Tripwire Purchase (10% = 30 buyers)
    ↓
Core Offer (20% = 6 buyers)
    ↓
High-Ticket (20% = 1-2 buyers)
```

- [ ] Calculate funnel metrics:
  - [ ] Lead Magnet conversion: Traffic → Leads (target: 20-40%)
  - [ ] Tripwire conversion: Leads → Buyers (target: 5-15%)
  - [ ] Core conversion: Tripwire → Core (target: 10-30%)
  - [ ] High-Ticket conversion: Core → High-Ticket (target: 10-30%)
- [ ] Calculate unit economics:
  - [ ] Cost per lead (CPL): Ad spend / leads
  - [ ] Cost per acquisition (CPA): Ad spend / Tripwire buyers
  - [ ] Customer lifetime value (LTV): Average revenue per customer
  - [ ] LTV:CAC ratio: Lifetime value / acquisition cost (target: 3:1+)
- [ ] Identify bottlenecks: Where is drop-off highest?
- [ ] Plan split tests: What to test first? (headline, price, guarantee)

### Step 8: Quality Check & Refinement (20-30 min)

- [ ] Review entire funnel for logical flow: Does each step lead naturally to next?
- [ ] Check value ladder logic: Does value increase at each level?
- [ ] Verify pricing: Is Tripwire impulse? Is Core profitable? Is High-Ticket premium?
- [ ] Test email sequences: Do they build trust and urgency?
- [ ] Validate follow-up system: Are we recovering non-buyers?
- [ ] Calculate projected ROI: Does the math work?
- [ ] Run through `checklists/funnel-quality-checklist.md` (if exists)

## Execution Notes

**Duration:** 4-6 hours for complete funnel architecture

**Complexity Levels:**

| Level | Funnel Structure |
|-------|-----------------|
| **Simple** | Lead Magnet → Core Offer (2-step) |
| **Standard** | Lead Magnet → Tripwire → Core (3-step) |
| **Advanced** | Lead Magnet → Tripwire → Core → High-Ticket (4-step) |
| **Full Stack** | Multiple lead magnets → segmented paths → multi-tier backend |

**Iteration Pattern:**
1. First pass: Map value ladder and core flow
2. Second pass: Design email sequences
3. Third pass: Calculate metrics and bottlenecks

**Common Pitfalls:**
- Too many steps (complexity kills conversion)
- Weak lead magnet (attracts wrong people)
- No ascension path (leave money on table)
- Poor follow-up (lose non-buyers forever)
- Overpriced Tripwire (kills impulse)

**Success Criteria:**
- Clear value progression (each step leads to next)
- Profitable unit economics (LTV > CAC by 3x+)
- High conversion rates at each step
- Sustainable traffic-to-revenue system
