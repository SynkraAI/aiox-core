# Checklist de Qualidade de Roteiros de Vídeo

Aplicar antes de aprovar qualquer roteiro destinado a Reels, TikTok, YouTube Shorts ou YouTube longo.

## Gancho de Abertura (Primeiros 3 Segundos)

- [ ] Os primeiros 1-3 segundos contêm o hook mais forte do roteiro
- [ ] A primeira linha falada OU o primeiro visual funcionam como pattern interrupt
- [ ] Não começa com introdução sobre quem é o criador ("Olá, eu sou...")
- [ ] Não começa com "Hoje vou te ensinar..." sem tensão ou promessa clara

## Estrutura Narrativa

- [ ] Segue uma estrutura reconhecível e aplicada conscientemente:
  - **CEQCOM:** Contexto → Estímulo → Questão → Conflito → Opção → Mensagem
  - **Chicote (Natanael):** Hook → Problema → Jornada → Revelação → CTA
  - **PSP:** Problema → Solução → Prova
  - Ou outra estrutura definida no brief do roteiro
- [ ] Estrutura escolhida está indicada no cabeçalho do roteiro
- [ ] Transições entre seções são suaves e mantêm o ritmo

## Timing e Ritmo

- [ ] Timing marks estão presentes no roteiro (ex: [0:00-0:03], [0:04-0:15])
- [ ] Duração total estimada está dentro do limite da plataforma-alvo
- [ ] Não há "buracos de ritmo" — seções longas sem ação ou transição
- [ ] Velocidade de fala é realista para o timing indicado

## CTA

- [ ] CTA presente e claro no roteiro (não implícito)
- [ ] CTA posicionado no momento certo (geralmente no final, às vezes no meio em vídeos longos)
- [ ] CTA adequado para a plataforma (ex: "link na bio" para Instagram, "inscreva-se" para YouTube)
- [ ] Não há CTAs em excesso que fragmentam a atenção

## Adaptação por Plataforma

- [ ] O roteiro indica explicitamente a plataforma-alvo
- [ ] Linguagem e ritmo estão calibrados para o formato (Reel é mais rápido que YouTube longo)
- [ ] Se o mesmo roteiro serve múltiplas plataformas, as variações estão documentadas
- [ ] Elementos nativos da plataforma estão indicados (ex: texto na tela, stickers, chapters)

## Linguagem e Naturalidade

- [ ] Linguagem conversacional — soa como fala, não como leitura
- [ ] Frases curtas e diretas predominam
- [ ] Evita palavras difíceis de pronunciar naturalmente
- [ ] Tom de voz alinhado com o brand voice

## Resultado Final

- [ ] **APROVADO** — Roteiro pronto para gravação
- [ ] **REVISÃO MENOR** — Ajustes de ritmo ou CTA necessários
- [ ] **REESCREVER** — Estrutura ou hook não funcionam, devolver para o agente responsável
