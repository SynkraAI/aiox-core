# Template de Post — Instagram / LinkedIn / X

> Define a estrutura de um post de texto para redes sociais. Hook → Body → CTA.

---

## Metadata

```yaml
template_id: post-tmpl
format: post
platform: [instagram, linkedin, x]
agent: nicolas-cole, justin-welsh, natanael-oliveira
method: atomic-essay + content-os + copy-andromeda
```

---

## Structure

```
---
PLATAFORMA: {PLATFORM}
PILAR: {CONTENT_PILLAR}        # ex: autoridade / educação / comunidade / entretenimento
TEMA: {TOPIC}
SENTIMENTO-ALVO: {EMOTION}     # ex: curiosidade / dor / aspiração / urgência
---

[HOOK — 1 linha, máx 15 palavras]
{HOOK_LINE}

[BODY — 3 a 7 parágrafos curtos, 1-2 linhas cada]
{BODY_PARAGRAPH_1}

{BODY_PARAGRAPH_2}

{BODY_PARAGRAPH_3}

{BODY_PARAGRAPH_4}  ← opcional

{BODY_PARAGRAPH_5}  ← opcional

[VIRADA / INSIGHT]
{KEY_INSIGHT}

[CTA — 1 linha, ação clara]
{CTA_LINE}

[HASHTAGS — somente Instagram]
{HASHTAG_1} {HASHTAG_2} {HASHTAG_3} ← máx 5 hashtags relevantes
```

---

## Usage Notes

- O hook deve funcionar sem contexto — o leitor não sabe nada ainda
- Cada parágrafo do body deve ter uma ideia única (Atomic Essay: 1 post = 1 ideia)
- O CTA deve pedir UMA ação, nunca duas
- Espaçamento entre parágrafos é obrigatório para leitura mobile
- Não use bullet lists no Instagram — texto corrido funciona melhor

## Platform Adaptations

| Platform  | Adjustment                                                                 |
|-----------|----------------------------------------------------------------------------|
| Instagram | Até 2.200 chars. Hook visível antes do "mais". Hashtags no final (máx 5). |
| LinkedIn  | Até 3.000 chars. Primeira linha = hook completo. Sem hashtags em excesso.  |
| X         | Até 280 chars por tweet. Adaptar para thread se necessário.                |

## Hook Templates (by emotion)

| Emoção      | Modelo de Hook                                      |
|-------------|-----------------------------------------------------|
| Curiosidade | "A maioria das pessoas não sabe que {FATO}."        |
| Dor         | "Você está {AÇÃO_ERRADA} e nem percebe."            |
| Aspiração   | "Como eu {RESULTADO_DESEJADO} sem {OBSTÁCULO}."     |
| Autoridade  | "Depois de {TEMPO/NÚMERO}, aprendi que {LIÇÃO}."    |
| Urgência    | "Isso vai mudar em {PRAZO}. Leia agora."            |
```
