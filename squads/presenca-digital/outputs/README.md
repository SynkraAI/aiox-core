# Outputs — Presença Digital Squad

This directory contains runtime artifacts generated during squad execution.

## Structure

```
outputs/
├── content/           # Generated content pieces
│   ├── posts/         # Written posts by platform
│   ├── carousels/     # Carousel content
│   ├── scripts/       # Video scripts (reels, YouTube, shorts)
│   ├── blog/          # Blog post drafts
│   └── newsletters/   # Newsletter editions
├── calendars/         # Content calendars
├── campaigns/         # Weekly campaign artifacts
├── distribution/      # Distribution plans
└── repurposing/       # Repurposing maps and derivatives
```

## Naming Convention

`{YYYY-MM-DD}-{format}-{topic-slug}.md`

Example: `2026-03-10-reel-script-como-precificar.md`
