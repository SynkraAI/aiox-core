## 🎨 Features Obsidian Utilizadas

### Wikilinks
- `[[Conceito]]` - Navegação entre notas
- `[[MOC - Módulo X]]` - Voltar ao módulo
- `[[Arquivo#Secao]]` - Link para seção específica

### Graph View
- Visualização de conexões entre conceitos
- Identificação de conceitos centrais
- Navegação visual

### Backlinks
- Cada conceito mostra onde é referenciado
- Útil para ver impacto de um conceito

### Tags
- `#modulo-1` a `#modulo-5` - Filtrar por módulo
- `#abertura`, `#promocao`, `#entrega`, `#historia`, `#finalizacao` - Por categoria

### Templates
- Sistema de templates integrado
- Variáveis substituíveis
- Reutilização de estrutura

## 📱 Como Usar

### 1. Abrir o Vault
```bash
# Baixar Obsidian (gratuito)
https://obsidian.md

# Abrir Obsidian
# "Open folder as vault" → selecionar obsidian-vault/
```

### 2. Navegação Recomendada
1. Comece pelo `README.md`
2. Vá para `MOCs/` para visão geral
3. Use graph view (Ctrl/Cmd + G) para ver conexões
4. Clique em wikilinks `[[Conceito]]` para navegar

### 3. Preparar Aula
1. Abra o MOC do módulo
2. Revise conceitos-chave
3. Use templates para criar materiais customizados
4. Consulte "Notas do Instrutor"
