# Retenção — Anatomia, Diagnóstico e Tratamento

## Anatomia da Retenção

Retenção é a % do vídeo que a audiência assiste. Cada segundo do vídeo precisa justificar o próximo — se parar de entregar valor, a pessoa sai.

### Momentos Críticos

```
0-3s:   HOOK — Se passar, 70% assistem mais
3-10s:  SETUP — Estabelecer valor, inserir open loop
10-30s: VALUE — Entregar a promessa do hook
30-50s: PEAK — Melhor parte do vídeo
50-60s: PAYOFF + CTA — Satisfação + ação
```

### Benchmarks

| Métrica | Ruim | OK | Bom | Viral |
|---------|------|----|-----|-------|
| Retention @3s | < 60% | 60-70% | 70-80% | > 80% |
| Watch-through | < 25% | 25-35% | 35-50% | > 50% |
| Replays | < 2% | 2-5% | 5-10% | > 10% |

## Tipos de Drop-off e Tratamento

### 1. Hook Cliff (0-3s) — CRÍTICO

**Sintoma:** Queda > 30% nos primeiros 3 segundos.

**Causa:** Hook fraco. Sem pattern interrupt, sem curiosidade, talking head estático.

**Tratamento:**
- Substituir hook (ver references/hooks.md)
- Adicionar movimento visual nos primeiros 0.5s
- Audio cue no segundo 0 (scratch, swoosh, alert)

### 2. Setup Cliff (5-15s)

**Sintoma:** Queda > 15% entre os segundos 5-15.

**Causa:** Setup muito longo. Enrolação sem entregar valor. Sem open loop.

**Tratamento:**
- Cortar setup para < 8s
- Inserir open loop antes do segundo 10 ("mas o erro 3 vai te chocar")
- Primeiro valor entregue antes do segundo 8

### 3. Value Cliff (15-30s)

**Sintoma:** Queda constante na seção de valor.

**Causa:** Promessa do hook não sendo cumprida. Conteúdo genérico.

**Tratamento:**
- Entregar valor mais específico (números, exemplos reais)
- Adicionar prova social (before/after, dados)
- Inserir save trigger ("anota esses passos")

### 4. Transition Cliff

**Sintoma:** Queda abrupta entre seções.

**Causa:** "Vale" visual/narrativo — momento sem novidade onde a audiência interpreta como "acabou".

**Tratamento:**
- Transição dinâmica (motion + audio cue) em < 1s
- Texto overlay com open loop ("mas o pior é esse 👇")
- Mudança de enquadramento (close-up → medium shot)

### 5. CTA Cliff (últimos 5-10s)

**Sintoma:** Queda abrupta no final.

**Causa:** CTA abrupto ou genérico. "Gostou? Curte e segue!" não retém.

**Tratamento:**
- Transição suave para CTA (não cortar conteúdo → CTA)
- CTA específico ("qual desses você vai tentar? comenta")
- Save trigger antes do CTA ("salva pra não esquecer")

## Open Loops (Técnica de Retenção)

Open loop = pergunta sem resposta que a mente precisa fechar.

```
Segundo 3: "3 erros que todo mundo comete..."  (quais são?)
Segundo 15: "Mas o erro 3 vai te chocar..."     (qual é?)
Segundo 30: "E o resultado disso foi..."        (o quê?)
```

**Regra:** Mínimo 2 open loops por vídeo de 30-60s. Espaçar a cada 10-15s.

## Re-watch Triggers

Elementos que fazem a pessoa assistir de novo (replay = sinal forte para o algoritmo):

1. **Informação densa** — Dados que não dá pra absorver de uma vez
2. **Easter egg visual** — Detalhe que só percebe na segunda vez
3. **Velocidade** — Informação rápida demais → replay para anotar
4. **Plot twist** — Final que muda o contexto do início → replay para "pegar"
