# Hooks — Fórmulas Testadas para Parar o Scroll

## Os Primeiros 3 Segundos

```
0-0.5s: Visual impact (para o scroll)
0.5-1.5s: Curiosity activation (quer saber mais)
1.5-3s: Value promise (vai valer a pena assistir)
```

Se passar dos 3s, 70% da audiência continua. Se perder nos 3s, o vídeo morre.

---

## 10 Tipos de Hook (ranqueados por performance)

### 1. Pattern Interrupt (Retention: 82%)

**Fórmula:** Ação/visual inesperado + statement contraditório

```
Copy: "Você está fazendo [X] ERRADO"
Visual: Ação dramática (jogar comida fora, parar exercício)
Audio: Scratch sound + silêncio
```

**Por que funciona:** Quebra expectativa → cérebro precisa processar → para o scroll.

### 2. Before/After Split (Retention: 85%)

**Fórmula:** Transformação visual dramática + timeline

```
Copy: "90 dias atrás vs hoje"
Visual: Split screen lado a lado
Audio: Build-up music
```

**Por que funciona:** Prova social visual instantânea. Impossível não querer saber como.

### 3. Curiosity Gap (Retention: 78%)

**Fórmula:** Tease de informação valiosa + mistério

```
Copy: "Ninguém te conta isso sobre [X]..."
Visual: Pessoa sussurrando, tom íntimo
Audio: Whisper ASMR + suspense
```

**Por que funciona:** Cria "lacuna" de conhecimento que a mente quer fechar.

### 4. Bold Claim (Retention: 80%)

**Fórmula:** Afirmação que contradiz crença comum

```
Copy: "[Crença popular] na verdade é [oposto]"
Visual: Visual de contradição
Audio: Dramatic reveal sound
```

**Por que funciona:** Dissonância cognitiva → "será que é verdade?" → precisa assistir.

### 5. Number Shock (Retention: 75%)

**Fórmula:** Estatística surpreendente + bold text

```
Copy: "[Número grande]% das pessoas fazem isso e não sabem"
Visual: Número grande na tela
Audio: Alert sound
```

**Por que funciona:** Números específicos são mais credíveis que afirmações vagas.

### 6. Question Hook (Retention: 72%)

**Fórmula:** Pergunta que a audiência quer responder

```
Copy: "Por que você não consegue [resultado desejado]?"
Visual: Frustração relatable
Audio: Pensativo, curioso
```

### 7. Timeline Tease (Retention: 76%)

**Fórmula:** O que acontece ao longo do tempo

```
Copy: "O que acontece quando você [ação] por [tempo]"
Visual: Timeline gráfica
Audio: Build-up progressivo
```

### 8. Controversy Lite (Retention: 83%)

**Fórmula:** Opinião polêmica mas defensável

```
Copy: "[Coisa popular] é overrated. Sério."
Visual: Expressão desafiadora
Audio: Dramatic pause
```

**Cuidado:** Controvérsia gera engagement MAS pode alienar. Usar com moderação.

### 9. POV/Relatable (Retention: 70%)

**Fórmula:** Situação que todo mundo vive

```
Copy: "POV: Você no primeiro dia de [X]"
Visual: Encenação relatable
Audio: Trending audio (se houver)
```

### 10. Secret/Hack (Retention: 77%)

**Fórmula:** Informação "exclusiva"

```
Copy: "[Profissionais] não querem que você saiba isso"
Visual: Tom conspirativo, close-up
Audio: Sussurro + reveal
```

---

## Regras Gerais de Hook

1. **Movimento nos primeiros 0.5s** — Scroll para na mudança visual
2. **Copy curta** — Máximo 10 palavras no hook
3. **Uma ideia por hook** — Não misturar tipos
4. **Testar, não adivinhar** — A/B test com dados reais
5. **Combinar é poderoso** — Pattern Interrupt + Curiosity Gap = 90%+ retention
