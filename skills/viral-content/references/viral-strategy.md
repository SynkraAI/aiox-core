# Estratégia Viral — Frameworks e Psicologia

## STEPPS Framework (Jonah Berger)

O framework mais validado para avaliar potencial viral. Cada elemento é um "motor" de compartilhamento.

| Elemento | O que é | Pergunta-chave | Exemplo |
|----------|---------|----------------|---------|
| **S**ocial Currency | Compartilhar faz a pessoa parecer inteligente/conectada | "Quem compartilha isso parece _____?" | "Descobri uma técnica que 90% não conhece" |
| **T**riggers | Algo no dia a dia que lembra o conteúdo | "Quando a pessoa vai lembrar disso?" | Vídeo sobre café → lembra todo dia de manhã |
| **E**motion | Ativa emoção forte (surpresa, raiva, alegria, medo) | "A pessoa SENTE algo forte?" | Before/after de transformação = inspiração |
| **P**ublic | Visível — outras pessoas veem quem compartilha | "Dá pra ver que alguém compartilhou?" | Tag de amigo, duet, stitch |
| **P**ractical Value | Útil — a pessoa quer guardar pra usar depois | "Alguém vai SALVAR isso?" | Checklist, tutorial, receita |
| **S**tories | Embala valor dentro de uma narrativa | "Tem uma história por trás?" | "Eu fazia errado por 5 anos até que..." |

### Como usar

Para cada conceito de vídeo, avaliar cada elemento de 0-10. Score total > 40 = bom potencial viral. Score > 50 = excelente.

## Hook-Retain-Convert (H-R-C)

Framework de estrutura de vídeo viral em 3 blocos:

```
[0-3s] HOOK — Parar o scroll
  - Pattern interrupt (visual ou auditivo)
  - Curiosity gap (criar mistério)
  - Promessa de valor
  Métrica: Retention @3s > 75%

[3s-fim] RETAIN — Manter assistindo
  - Open loops (perguntas sem resposta)
  - Value delivery (cumprir promessa)
  - Tension/release cycles
  - Pacing (nunca mais que 3s sem novidade)
  Métrica: Watch-through > 40%

[últimos 5s] CONVERT — Gerar ação
  - CTA claro e específico (não genérico)
  - Save trigger ("salva pra depois")
  - Share trigger ("marca quem precisa ver")
  - Follow trigger ("sigo pra mais")
  Métrica: Save rate > 8%, Share rate > 3%
```

## Share Psychology

Por que as pessoas compartilham conteúdo:

1. **Identity Expression** — "Compartilhar isso mostra quem eu sou"
   - Ex: "Isso é TÃO eu" → share
2. **Social Validation** — "Compartilhar isso me dá status"
   - Ex: "Encontrei primeiro" → share
3. **Practical Value** — "Isso ajuda alguém que eu conheço"
   - Ex: "Meu amigo PRECISA ver isso" → share
4. **Emotional Arousal** — "Senti algo forte e preciso dividir"
   - Ex: "NÃO ACREDITO nisso" → share

### Share Triggers (inserir no vídeo)

- **Tag trigger**: "Marca aquele amigo que faz isso" (obrigação social)
- **Save trigger**: "Salva pra não esquecer" (valor prático)
- **Debate trigger**: "Concorda ou discorda? Comenta" (Social Currency)
- **Identification trigger**: "Se você faz [X], precisa ver isso" (identidade)

## Viral Loop Design

```
AQUISIÇÃO → O vídeo alcança novas pessoas (algoritmo)
     ↓
ATIVAÇÃO → Pessoa assiste (hook funciona)
     ↓
RETENÇÃO → Pessoa assiste até o final (conteúdo entrega valor)
     ↓
REFERRAL → Pessoa compartilha (share trigger funciona)
     ↓
RECEITA → Pessoa segue / salva / visita perfil
     ↓
(loop recomeça com o share ampliando o alcance)
```

**Viral Coefficient**: Se cada viewer gera > 1 novo viewer = crescimento exponencial.
- Coefficient < 1.0 = conteúdo morre
- Coefficient 1.0-1.5 = crescimento linear
- Coefficient > 1.5 = viral
- Coefficient > 3.0 = mega viral

## Script Frameworks

### PAS (Problem-Agitate-Solution)

```
[0-10s]  PROBLEM: Identificar dor
[10-30s] AGITATE: Tornar a dor pior
[30-50s] SOLUTION: Apresentar solução
[50-60s] CTA: Chamar para ação
```

### BAB (Before-After-Bridge)

```
[0-10s]  BEFORE: Estado atual (dor)
[10-30s] AFTER: Estado desejado (transformação)
[30-50s] BRIDGE: Como chegar lá
[50-60s] CTA
```

### AIDA

```
ATTENTION: Hook (0-3s)
INTEREST: Contexto que gera curiosidade (3-15s)
DESIRE: Mostrar resultado possível (15-40s)
ACTION: CTA (40-60s)
```
