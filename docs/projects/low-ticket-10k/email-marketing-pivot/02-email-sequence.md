# 02 — Sequência de Email Completa (7 emails, D+0 → D+10)

**Estrutura:** Soap Opera Sequence (Brunson) + Awareness Progression (Wiebe) + Voice (Handley) + Hard CTAs justificados (Hormozi) + Permission base (Godin).

**Variante apresentada:** **Versão ADVOGADO** (template). User adapta para MEI e Professor trocando contexto, exemplos e dores específicos. Estrutura permanece idêntica.

**Convenções:**
- Subject lines: 3 variantes A/B/C (testar com 33% cada)
- Preview text: complementa subject (não repete)
- Corpo: markdown — ao migrar pro Resend, converter para HTML simples ou usar template Resend
- Cada email tem 1 CTA principal + 1 PS (regra Brunson)

americano + jurisdições aleatórias).

Conserto: dá CONTEXTO MÍNIMO em 4 itens — papel,

prompt como RASCUNHO 80% pronto, não como produto
final.

- PS sobre garantia constrói confiança (anti-Hormozi de "garantia inversa", mas Wiebe wins aqui pro mercado BR mais conservador)
