Política de Privacidade

Última atualização: 27 de março de 2026

A VORZA ("nós", "nosso") opera o site e os produtos digitais associados. Esta página informa sobre nossas políticas relativas à coleta, uso e divulgação de dados pessoais quando você utiliza nossos serviços.

1. Dados Coletados

Coletamos os seguintes dados:

  • Dados de compra: Nome, email, CPF (para emissão de nota fiscal), coletados pela plataforma Kiwify durante o checkout.
  • Dados de navegação: Cookies, IP, tipo de navegador, páginas visitadas, coletados automaticamente para fins de análise e otimização.
  • Pixel de rastreamento: Utilizamos o Meta Pixel (Facebook) para medir a eficácia de anúncios. Este pixel coleta dados anonimizados de navegação.

2. Uso dos Dados

Seus dados são utilizados exclusivamente para:

  • Processar sua compra e entregar o produto digital;
  • Enviar comunicações relacionadas ao produto (atualizações, suporte);
  • Melhorar nossos produtos e experiência do usuário;
  • Cumprir obrigações legais e fiscais.

3. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto:

  • Kiwify: Plataforma de pagamento que processa sua compra;
  • Meta (Facebook): Dados anonimizados via Pixel para otimização de anúncios;
  • Google: Dados anonimizados via Google Analytics (se aplicável);
  • Autoridades legais: Quando exigido por lei.

4. Cookies

Utilizamos cookies essenciais para o funcionamento do site e cookies de análise para entender como os visitantes interagem com nossas páginas. Você pode desabilitar cookies nas configurações do seu navegador, mas isso pode afetar a funcionalidade do site.

5. Segurança

Empregamos medidas de segurança razoáveis para proteger seus dados. O processamento de pagamentos é feito integralmente pela Kiwify, que utiliza criptografia SSL/TLS e está em conformidade com padrões PCI-DSS.

6. Seus Direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento de seus dados;
  • Acessar seus dados pessoais;
  • Corrigir dados incompletos ou desatualizados;
  • Solicitar a exclusão de seus dados pessoais;
  • Revogar consentimento a qualquer momento.

Para exercer qualquer desses direitos, entre em contato: suportevorza@gmail.com

7. Retenção de Dados

Seus dados de compra são retidos pelo período necessário para cumprir obrigações fiscais (5 anos conforme legislação brasileira). Dados de navegação são retidos por até 12 meses.

8. Alterações

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas por email.

9. Contato
