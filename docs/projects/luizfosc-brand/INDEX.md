## Decisões Tomadas

### Tagline
- **"Não é dom, é estrutura."** (SMILE 45/50, SCRATCH 10/70)
- "Inspirando. Sempre." — aposentada

### Categoria
- **Sem categoria. Só declaração.** Modelo Steve Jobs.
- Bio: `Luiz Fosc · Não é dom, é estrutura. · 2× Guinness · #pensecomoumilusionista`

### Manifesto
- Mix de 3 tons: provocativo + cinematográfico + vulnerável

### Movimento
- #pensecomoumilusionista com 6 Princípios do Ilusionista
- Inimigo: O Mito do Dom
- Filosofia core: criar experiências como a Disney cria mundos

### Manifesto — APROVADO
- Versão A "Provocação com Alma" — mix 3 tons (provocativo + cinematográfico + vulnerável)
- Arquivo: `docs/research/2026-04-07-FOSC-ai-brand-identity-tools/manifesto-aprovado.md`
- Feedback: "menos é mais" — informações pessoais são contexto, não conteúdo

### Visual — APROVADO (Sessão 5)
- **Logo:** Manter a existente (diamante/chevron geométrico + Montserrat)
- **Paleta base:** B&W (#212121 + #FFFFFF) + Amber (#B8860B) — permanente
- **Cor de humor:** Teal (#1A3A3A) — modo editorial/reflexivo
- **Tipografia:** Montserrat (títulos, logo) + Ubuntu (body, quotes) — evolução do manual 2019
- **Padrão gráfico:** Chevron pattern (5 usos: logo, bg, corner, seta, highlight icons)
- **Elementos:** Moldura retangular (teal/amber), aspas " grandes em amber, símbolo cortado na borda
- **5 Humores:** Dramático, Editorial, Humano, Energia, Premium
- **Cores de campanha:** Temporárias por produto (Deep Red, Burnt Orange, Forest Green, Steel Blue, Warm Brown)
- **Anti-AI:** Direção 100% baseada em DNA real (manual 2019 + Beace + kit Canva), zero tendência
- **Palestra "Fora da Caixa":** Electric Amber #E8A317, cubo aberto, layout assimétrico, "QUE CAIXA?"

### Objetivo Real
- Vender mais palestras (R$35K+) para decisores C-level/RH
- Posicionar Mentoria Palestra de Elite para profissionais que querem virar palestrantes
