Politica de Privacidade

Ultima atualizacao: 10 de abril de 2026

1. Introducao

A Bretda ("nos", "nosso") opera o site bretda.com.br e aplicacoes internas de gerenciamento de campanhas publicitarias. Esta Politica de Privacidade descreve como coletamos, usamos e protegemos suas informacoes ao utilizar nossos servicos.

2. Informacoes que Coletamos

Nosso aplicativo acessa dados de plataformas de publicidade (Google Ads e Meta Ads) exclusivamente para fins de gerenciamento de campanhas. Os dados acessados incluem:

  • Metricas de desempenho de campanhas (impressoes, cliques, custos, conversoes)
  • Configuracoes de campanhas, grupos de anuncios e palavras-chave
  • Dados demograficos agregados de audiencia
  • Informacoes de conta publicitaria (ID da conta, nome)

Nao coletamos dados pessoais de usuarios finais das campanhas publicitarias.

3. Como Usamos as Informacoes

As informacoes acessadas sao utilizadas exclusivamente para:

  • Monitoramento e analise de desempenho de campanhas
  • Otimizacao de configuracoes de campanhas publicitarias
  • Geracao de relatorios internos de performance
  • Gerenciamento de orcamentos e lances

4. Armazenamento e Seguranca

Os dados acessados sao processados localmente em sistemas internos protegidos. Implementamos medidas de seguranca tecnicas e organizacionais para proteger as informacoes, incluindo:

  • Credenciais armazenadas de forma segura com variaveis de ambiente
  • Acesso restrito apenas a pessoal autorizado
  • Tokens de autenticacao com renovacao periodica
  • Comunicacao criptografada via HTTPS/TLS

5. Compartilhamento de Dados

Nao compartilhamos, vendemos ou transferimos dados de campanhas publicitarias para terceiros. Os dados sao utilizados exclusivamente para os fins descritos nesta politica.

6. Uso de APIs do Google

Nosso aplicativo utiliza a API do Google Ads para acessar dados de campanhas. O uso e a transferencia de informacoes recebidas de APIs do Google para qualquer outro aplicativo estao em conformidade com a Politica de Dados do Usuario dos Servicos de API do Google, incluindo os requisitos de Uso Limitado.

7. Retencao de Dados

Os dados de campanhas sao mantidos apenas pelo periodo necessario para analise e otimizacao. Dados historicos de desempenho podem ser mantidos para fins de comparacao e relatorios, mas podem ser excluidos a qualquer momento mediante solicitacao do titular da conta.

8. Seus Direitos

Em conformidade com a Lei Geral de Protecao de Dados (LGPD), voce tem direito a:

  • Solicitar acesso aos dados processados
  • Solicitar a exclusao dos dados armazenados
  • Revogar o consentimento de acesso a qualquer momento
  • Solicitar informacoes sobre o tratamento de dados

9. Revogacao de Acesso

Voce pode revogar o acesso do nosso aplicativo aos seus dados do Google Ads a qualquer momento atraves das configuracoes de seguranca da sua conta Google.

10. Contato

Para duvidas ou solicitacoes relacionadas a esta Politica de Privacidade, entre em contato:

  • Email: contato@bretda.com.br
  • Site: bretda.com.br

11. Alteracoes nesta Politica

Podemos atualizar esta Politica de Privacidade periodicamente. Alteracoes significativas serao comunicadas atraves do nosso site. A data da ultima atualizacao e indicada no topo deste documento.
