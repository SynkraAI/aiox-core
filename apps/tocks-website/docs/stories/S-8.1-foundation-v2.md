# Story 8.1 — Foundation v2: Tokens Gilded Noir + Type Stack + Reduced Motion

## Story

**COMO** dev agent do Tocks Website v2
**QUERO** evoluir o design system Gilded Noir para v2 (noir 3-shade, bone layer editorial, gold 5-step, type stack Fraunces/Inter/Space Grotesk, `prefers-reduced-motion` global, cinematic shadows, paper-grain texture) preservando aliases legacy
**PARA** que as stories S-8.2 (hero), S-8.3 (provenance), S-8.4 (atelier longform), S-8.5 (motion) e S-8.6 (QA) possam consumir uma fundação cinematográfica sem regredir nenhum componente já entregue na Fase 7.

## Context

- **Direção aprovada 2026-04-17:** B (Gilded Noir Cinemático) + cherry-pick E (Provenance card Tufteano) + cherry-pick C (Atelier longform editorial).
- **Spec consumível:** `docs/design/elevation-awwwards/05-tokens-v2-spec.md` (Nova ratificou, cola-direto em `globals.css`).
- **Blocker WCAG Sessão 19 (Patch #2c):** `prefers-reduced-motion` NUNCA foi aplicado. S-8.1 resolve **antes** de qualquer motion nova entrar (S-8.5).
- **Estado atual `globals.css`:** 180 linhas, 3 cores gold (sem 5-step), 1 shade noir (sem deep/elevated), zero bone layer, zero texture, zero cinematic-shadow, tipografia Cormorant/Inter/Montserrat (Spiekermann vetou Cormorant commodity 2019-2024).
- **brad-frost (atomic ordering):** tokens primeiro, atoms herdam v2 automaticamente via CSS vars, molecules/organisms só migram quando atom migrou — esta story cobre camada token + swap de fontes; migração componente-a-componente entra nas stories subsequentes.
- **Constitution Art. IV (No invention):** não inventar tokens fora do spec 05. Não inventar variantes tipográficas além de Fraunces/Inter/Space Grotesk.
- **Constitution Art. VII:** arquivos tocados mantêm <100 linhas. `globals.css` é exceção (file-type CSS, não componente).
- **Next.js 16 AGENTS.md guardrail:** leitura obrigatória de `node_modules/next/dist/docs/` antes de mexer em `next/font/google` ou fonts layout — verificar axes/variable-font support.

### Val-head ratification (consultation 230aaf35)

- **Patch #2c is WCAG 2.2 AA blocker** — toda animação CSS `entrance-fade-up*` + `fade-in-scroll` + `shimmer` precisam desabilitar sob `prefers-reduced-motion: reduce`. Global `animation-duration: 0.01ms !important` + override explícito dos `.hero-*` resolve.

### Regra WCAG gold-on-bone (documentar em qa log)

- `--gold-500 (#D4AF37)` sobre `--bone-paper (#F5F0E6)` = 2.37:1 — **FALHA AA texto (requer 4.5:1)**.
- `--gold-500` sobre `--noir-mid (#0B0B0F)` = 9.03:1 — **PASS AAA**.
- `--bone-ink (#1A1812)` sobre `--bone-paper` = 15.2:1 — **PASS AAA**.
- **Regra para S-8.3 (provenance card) + S-8.4 (atelier longform):** em superfícies bone, gold apenas em hairlines (`border-color`), icon strokes, bullets decorativos. Texto sobre bone usa `--bone-ink`. Este guardrail é input obrigatório.

### Input para stories dependentes

- **S-8.2 Hero video:** consome `--shadow-cinematic` em container video + `.heading-display` na H1 (negative tracking Fraunces).
- **S-8.3 Provenance card:** consome `--bone-paper` bg + `--bone-ink` text + `--gold-700` hairlines + `.texture-paper` overlay.
- **S-8.4 Atelier longform:** consome `.section-bone`, `.container-editorial`, `.section-padding-editorial`, `--bone-rule` separators.
- **S-8.5 Motion system:** depende de Patch #2c aplicado — motion classes novas não entram até este Patch estar live.
- **S-8.6 QA:** valida contraste regra gold-on-bone documentada + `prefers-reduced-motion` cumprido em todas animações.

### Riscos identificados

- **R-1 (baixo):** Fraunces variable font load pode regredir LCP ≤ 100ms vs Cormorant. Mitigação: `display: 'swap'` + `axes` scoped. Validar via Lighthouse T-4.5.
- **R-2 (baixo):** Espaçamento tipográfico Fraunces vs Cormorant muda altura visual de H1/H2 — pode desalinhar por ~4-8px. Mitigação: `--tracking-headline: -0.02em` compensa. Smoke visual T-4.1 captura regressão.
- **R-3 (médio):** Alias legacy mal mapeado quebra um componente silenciosamente (ex: `--accent-ivory` apontando wrong). Mitigação: smoke visual T-4.1 em 5 páginas.
