# Story S-9.3 — R3F Foundation (cube placeholder pipeline)

## Scope

Instalar `three`, `@react-three/fiber`, `@react-three/drei` (tree-shaken subset: `OrbitControls`, `Environment`, `useGLTF`). Criar `src/components/organisms/configurator-3d.tsx` dinâmico via `dynamic(() => import('...'), { ssr: false })` em `/colecao/[slug]`. Cube-placeholder GLB (drei `<Box />` ou stock cube.glb) valida pipeline sem GLB real ainda. HDRI Poly Haven CC0 placeholder. Setup Vercel Blob upload script `scripts/upload-assets.ts` (CLI puro per art. I). Bundle analyzer gerado: **MUST fit ≤300 KB gzipped delta** vs baseline.

## Acceptance Criteria (draft)

1. **AC-1** — Deps instaladas: `three@^0.168`, `@react-three/fiber@^9`, `@react-three/drei@^10` (React 19 compatible). Package-lock commitado.
2. **AC-2** — `Configurator3D` organism ≤100L (Art. VII); cube-placeholder mesh com auto-rotate OFF default, OrbitControls com `enableDamping={true}` easing quartic-equivalent.
3. **AC-3** — Dynamic import isola R3F: `/` e `/colecao` first-load JS inalterado (≤+1 KB delta). `/colecao/[slug]` first-load ≤ 350 KB gzipped (ADR-003).
4. **AC-4** — Reduced-motion: `prefers-reduced-motion: reduce` → `auto-rotate` desabilitado (mesmo se futuro user-toggle implementado).
5. **AC-5** — Fallback poster: enquanto `useGLTF` carrega, `<ImagePlaceholder>` visible; `<Suspense fallback={...}>` wrapper.
6. **AC-6** — Vercel Blob: `scripts/upload-assets.ts` CLI uploads cube.glb + studio.hdr; URLs em `src/data/assets.ts`; `.env.local` BLOB_READ_WRITE_TOKEN documented em README.
7. **AC-7** — Bundle analyzer build output salvo em `docs/qa/s-9.3-bundle-before-after.html`; @architect ratifica delta.
