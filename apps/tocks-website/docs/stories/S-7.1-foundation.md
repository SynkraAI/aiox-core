# Story 7.1 — Foundation: Tokens Gilded Noir + Atoms + Data Updates

## Story

**COMO** dev agent do Tocks Website v2
**QUERO** estabelecer a fundação Gilded Noir (design tokens CSS ajustados + atoms base + dados normalizados)
**PARA** que todas as molecules, organisms e pages da Fase 2 possam consumir um sistema de tokens/atoms consistente, WCAG-AA-safe e alinhado ao catálogo real (8 mesas PT-BR).

## Context

- Plano SQUAD aprovado por `@ux-design-expert` com 3 ressalvas (ver Dev Notes).
- Blockers de sessão 9 TODOS resolvidos: products.ts PT-BR, WhatsApp/email em constants.ts.
- brad-frost (Atomic Design): "Design tokens são a fundação. Atoms depois. Molecules só quando atoms forem sólidos."
- Constitution Art. VII: max 100 linhas/componente — atoms DEVEM ser minimalistas.
- Plano SQUAD lista 2 atoms novos: `text-pair` (NOVO) e `step-indicator` (promovido a molecule, mas base atom-like em S-7.1 é descartada — step-indicator fica INTEGRALMENTE em S-7.2 como molecule).

**Decisão PM:** S-7.1 cria apenas `text-pair` (único atom puro novo). `step-indicator` vai integralmente para S-7.2 porque tem lógica ("X de Y") e pertence à camada molecule.

## Acceptance Criteria

1. **AC-1 (Tokens):** GIVEN `src/app/globals.css`, WHEN inspecionado, THEN contém tokens `--color-whatsapp-gold` (#C4A65E), `--color-whatsapp-gold-hover` (#D4B66F), `--color-focus-border: var(--color-accent-gold)` (full — NÃO `/40`), `--ring-focus-offset: 2px`, `--tracking-editorial: 0.2em`, `--tracking-extreme: 0.3em`. Token `--gradient-gold` REMOVIDO do arquivo.

2. **AC-2 (WCAG):** GIVEN qualquer token de focus-border aplicado sobre `bg-bone`, WHEN medido via contrast-ratio tool, THEN razão >= 3:1 (WCAG 2.2 AA para non-text UI).

3. **AC-3 (Atom text-pair):** GIVEN `src/components/atoms/text-pair.tsx`, WHEN renderizado com props `{label: string, value: string, as?: 'stat' | 'spec' | 'investimento'}`, THEN renderiza label em uppercase gold tracking-editorial + value abaixo em tipografia conforme variante. Arquivo < 100 linhas.

4. **AC-4 (Copy):** GIVEN `src/lib/constants.ts`, WHEN inspecionado, THEN `BRAND_COPY` contém `faq.headline = "Dúvidas Frequentes"` (nunca "Considerações do Atelier"); `projects.headline = "Projetos Realizados"`; todas strings revisadas sem ponto de exclamação; sem a palavra "endereço" (Atelier é brand concept, sem endereço público).

5. **AC-5 (Data):** GIVEN `src/data/products.ts`, WHEN cada produto inspecionado, THEN `customizationOptions` continua como array (2–4 opções/produto é o padrão real, NÃO forçar 7). Nenhuma mudança estrutural além de garantir consistência de strings PT-BR sem acentos fortemente-legacy.

6. **AC-6 (Art. VII):** GIVEN `atoms/text-pair.tsx`, WHEN wc -l executado, THEN resultado < 100 linhas.

7. **AC-7 (SSR-safe):** GIVEN atom `text-pair.tsx`, WHEN importado em Server Component, THEN não depende de `'use client'` (atom puro visual).
