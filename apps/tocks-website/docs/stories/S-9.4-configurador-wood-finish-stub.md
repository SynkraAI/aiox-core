# Story S-9.4 — Configurador: wood species + finish

## Scope

Substituir cube-placeholder por GLB real (peça 1 definida em Q1 Nova). UI de configurador: 3 wood swatches (freijó / ipê / jatobá) + 3 finish options (fosco / acetinado / alto-brilho). Swap material map em runtime — NÃO re-load GLB (caro); swap texture via `useTexture` + material.map reassignment, ou PBR preset via `material.roughness` update. ARIA announcement em mudança. Configurador recebe lead-gen button "Quero essa configuração" (Q5 Nova se YES) que gera WhatsApp URL com specs parametrizadas.

## Acceptance Criteria (draft)

1. **AC-1** — GLB real da peça 1 integrado via `useGLTF('/api/asset/peca1.glb')` (Vercel Blob URL do S-9.3 pipeline); render correto em Canvas.
2. **AC-2** — 3 wood swatches renderizam como atoms clicáveis (`WoodSwatch` component); clicks atualizam material diffuse map em runtime (< 50ms perception delay).
3. **AC-3** — 3 finish presets aplicam roughness/metalness deltas (fosco: roughness 0.9; acetinado: 0.5; alto-brilho: 0.1, metalness 0.3).
4. **AC-4** — `aria-live="polite"` announce: "Madeira alterada para freijó" / "Acabamento alterado para verniz fosco".
5. **AC-5** — Keyboard nav: Tab reach todos os 6 controls; ENTER seleciona; visible focus indicator gold on dark.
6. **AC-6** — Lead-gen (se Q5=YES): botão "Quero essa configuração" → `href='https://wa.me/{WHATSAPP_NUMBER}?text={urlEncoded(BRAND_COPY.configurator.whatsapp_template)}' + specs`.
7. **AC-7** — Reduced-motion: zero animation em swatch hover/selection além de Epic 8 baseline.
8. **AC-8** — Lighthouse `/colecao/[slug]` Perf ≥85 post-configurador-mount; LCP não conta Canvas render (is post-interactive).
