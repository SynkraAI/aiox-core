# Story S-9.5 — 360° rotate + ambient context

## Scope

Adicionar drei `<OrbitControls>` com `enableDamping`, min/max polar/azimuth angles constraining rotation sensata (não deixar user olhar mesa pelo chão). Auto-rotate opcional OFF default; toggle via small button "Rotação automática" (opt-in per val-head convention). `<Environment>` HDR studio (de S-9.3 Poly Haven CC0 OR commissioned). `<ContactShadows>` drei para floor shadow catcher (1 plano, opacity 0.4, scale tuned to GLB). Não adiciona room scene (Epic 10).

## Acceptance Criteria (draft)

1. **AC-1** — OrbitControls: `enableDamping={true}`, `dampingFactor={0.05}`, `maxPolarAngle={Math.PI * 0.55}` (não mostra underside), `minPolarAngle={Math.PI * 0.2}` (não top-down), `autoRotate` state-controlled.
2. **AC-2** — Auto-rotate toggle button (visible icon + label pt-BR "Rotação automática"); default OFF; `prefers-reduced-motion: reduce` **force** OFF ignorando state.
3. **AC-3** — `<Environment preset="studio" />` OR custom HDR blob URL; scene ambient intensity tuned com Nova (Gilded Noir chiaroscuro, não bright studio).
4. **AC-4** — `<ContactShadows opacity={0.4} scale={10} blur={2.5} far={2} />` drop-shadow sub-GLB.
5. **AC-5** — Perf budget: cena no idle ≤30 draw calls, ≤500k tris (bundle budget diff vs S-9.4 ≤+2 KB gzipped — HDR é runtime fetch, não bundle).
6. **AC-6** — WCAG: auto-rotate toggle é button com `aria-pressed={isRotating}`, focus visible, keyboard ENTER toggles.
7. **AC-7** — Screenshots salvos em `docs/qa/s-9.5-ambient-frames/` (5 ângulos × 3 wood × 3 finish = 45 frames para Nova ratification cromática).
