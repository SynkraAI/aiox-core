# Story 8.4 — Atelier Longform: 5 Chapter Templates + Broken Grid

## Story

**COMO** dev agent do Tocks Website v2
**QUERO** reescrever `src/app/atelier/page.tsx` como reportagem longform editorial estruturada em 5 chapters (Lead → Madeira → Corte → Acabamento → Entrega → Cólofon), criar 5 chapter templates broken-grid (`<chapter-target>`-pattern para GSAP scroll-storytelling), adicionar `variant="editorial"` ao `image-grid.tsx`, consumir `.section-bone` + `.container-editorial` da S-8.1
**PARA** que a página Atelier entregue o **cherry-pick C** (Ateliê Documental) da direção aprovada — storytelling longform NYT Magazine-level, broken grid wagneriano, memorabilidade FWA 9/10, sem sacrificar conversão (CTA concierge no final).

## Context

- **Direção aprovada:** B + **cherry-pick C** → `/atelier` vira longform editorial (NYT Magazine / The Gentlewoman / Apartamento).
- **Spec estrutura** (master-plan Fase 4b):
  - **Lead** (1 parágrafo): quem, onde, desde quando
  - **Ato 1 — Madeira:** foto full-bleed B&W + 3 parágrafos + sidebar small multiple origens (Tufte reference)
  - **Ato 2 — Corte:** broken grid 2 fotos + pull-quote marcenaria
  - **Ato 3 — Acabamento:** 4-up grid textural + legenda técnica
  - **Ato 4 — Entrega:** testimonial + CTA concierge
  - **Cólofon editorial:** ano, tiragem, catálogo anterior
- **Atelier atual** (`src/app/atelier/page.tsx` 127L): conteúdo razoável (PROCESS_STEPS + MATERIALS + TEAM) mas layout commodity grid. **REFAZER** (brief matriz).
- **Contraint Art. VII:** split em 5 templates ≤100 linhas cada.
- **INVEST check Will-Larson (autonomous ruling 9e69f721):** mantido single story porque:
  1. 5 chapter templates = estruturalmente idênticos (DRY via shared primitive `.chapter-target`), não são 5 stories.
  2. GSAP wiring é herdado de S-8.5 (Classe 3 Scroll Storytelling + `useChapterScroll` hook), S-8.4 apenas **consome** a class + hook.
  3. Content migration é text-constant, não lógica.
  4. Split 4a/4b introduziria merge hell em `image-grid.tsx` (ambos tocariam o organism).
  5. **Fallback documentado:** se @dev chega em 7h com >30% tasks restantes, cutover para split 4a (templates+grid) + 4b (content+wiring) — decisão reservada, não preemptive.
- **Broken grid:** `variant="editorial"` no `image-grid.tsx` atual (hoje só `project` + `related`). Assymmetric rows + caption off-axis + broken alignment.
- **Asset dependency:** Nano Banana 2 quota → fotos B&W atelier ainda placeholder. Entrega infra + copy + placeholder structure.
