## Binding rules for S-8.3 / S-8.4 / S-8.6

### RULE 1 — Gold on bone: **non-text only**

`--gold-500` over any bone surface (`--bone-paper`, `--bone-warm`) **fails AA for text** (2.37:1).

**Allowed usages of gold over bone:**
- Hairlines (`border-color`)
- Icon strokes (SVG stroke)
- Decorative bullets, ornaments, rules
- Divider lines
- Focus rings (≥3:1 non-text — passes)

**Forbidden usages of gold over bone:**
- Body paragraph text
- Heading text (any size)
- Link text
- Button label text on gold-on-bone buttons
- Stats numerals

### RULE 2 — Text on bone uses `--bone-ink`

Any text over `--bone-paper` or `--bone-warm` **must** use `--bone-ink` (15.2:1, AAA).
For emphasis/display, `--gold-900 #4A3A1E` is acceptable alternative (≥9.5:1).

### RULE 3 — Text on noir uses `--text-primary` or `--gold-500`

Over `--noir-deep` / `--noir-mid` / `--noir-surface` / `--noir-elevated`:
- Body: `--text-primary` (AAA)
- Secondary: `--text-secondary` (AAA)
- Accent: `--gold-500` (AAA at 9.03:1)
- Never: `--gold-700` on noir (contrast < 3:1 — visual noise)

### RULE 4 — Focus rings

`--gold-500` focus-ring over any surface:
- On noir: 9.03:1 PASS
- On bone: 2.37:1 — **fails AA text, but non-text UI threshold is 3:1** — STILL FAILS.
- **Action for S-8.3/S-8.4 on bone surfaces:** focus ring must shift to `--gold-700` (≥3:1 non-text on bone) or `--bone-ink` (AAA). Document per-component in story.

## How this log is consumed

- **S-8.3 (provenance card):** card bg `--bone-paper`, text `--bone-ink`, hairlines `--gold-700`, icons `--gold-700`. Focus shifts to `--gold-700` or `--bone-ink` when card on bone.
- **S-8.4 (atelier longform):** `.section-bone` bg, body text `--bone-ink`, pull quotes may use `--gold-900` for display, decorative rules `--bone-rule` or `--gold-700`.
- **S-8.6 (QA):** axe-core + manual contrast audit must enforce these four rules. Any gold text over bone = defect.
