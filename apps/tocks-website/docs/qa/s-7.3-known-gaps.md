- **Symptom:** any submit to `<form action={submitConcierge}>` on /contato returns:
  ```
  Error: A "use server" file can only export async functions, found object.
  See https://nextjs.org/docs/messages/invalid-use-server-value
  ```

- **Action plan (suggested, NOT applied — out of QA scope):**
  1. Create `src/app/actions/submit-concierge.types.ts` with the `ConciergeFormState` type and the `INITIAL_CONCIERGE_STATE` constant.
  2. Re-export the type from `submit-concierge.ts` if needed (`export type { ConciergeFormState } from './submit-concierge.types'`).
  3. Update `src/components/organisms/concierge-form.tsx` to import `INITIAL_CONCIERGE_STATE` from the new types module.
  4. Re-run `npx playwright test tests/qa/server-action.spec.ts`. Spec is written so both the known-failure branch and the post-fix branch will pass.
- **Blocks merge:** **YES**. Form is the primary CTA on /contato — shipping it broken would be an immediate revenue/lead leak.
